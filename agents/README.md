# Agents - 2体運用アーキテクチャ

このディレクトリでは、自律型AI開発を支える「実装担当」と「監査担当」の2種類のエージェント設定とスキルを管理します。

## 2体運用のコンセプト：ガバナンスと分業

1台のAIにすべてを任せてはならない、というのが本プレイブックの核心です。役割を物理的に分けることで、人間が介在しなくても高いコード品質と設計の整合性を維持します。

### エージェントの役割分担

| エージェント | 役割 (Role) | 権限 |
|---|---|---|
| **Claude Code** | 実装 (Doer) | src/ 書き込み可 |
| **Gemini CLI** | 監査 (Auditor) | 読み取り専用 / docs/ のみ |
