# ブラウザ完結のビジュアルノベル制作プラットフォーム「kaedevn」、無料公開を開始

個人開発者の村久長 健太郎は、2026年X月X日、ビジュアルノベルの制作・公開・プレイをブラウザだけで完結できるプラットフォーム「kaedevn（かえで）」の無料公開を開始しました。

---

## 概要

kaedevn は、プログラミング不要のブロックエディタとプロ向けスクリプト言語を搭載したビジュアルノベル制作ツールです。作った作品はURLを共有するだけで、PCでもスマホでもインストールなしでプレイできます。

マルチプラットフォーム対応のネイティブエンジン（SDL2 + C++）も開発済みで、Android・iOS・macOSでの動作テストを進めています。Nintendo Switch への展開も計画しています。

## 背景

ビジュアルノベル制作の既存ツールは「PCアプリのインストールが必要」「コンソール展開に別エンジンが必要」という課題があります。kaedevn はブラウザ上の制作ツールとコンソール対応エンジンの両方を1つのプラットフォームに統合し、この課題に対応します。

## 主な機能

**制作**
- ブロック（GUI）とスクリプト（テキスト）が双方向同期するハイブリッド・エディタ。13種類のブロック
- タイムラインアニメーション、15種のスクリーンフィルター（GLSL）
- ファンタジー・現代学園の背景・立ち絵・BGM・SEを標準搭載

**プレイ**
- ブラウザで即プレイ、PWA対応、スマホタッチジェスチャー対応
- 20枠セーブ、クラウドセーブ、オート・スキップ、ギャラリー、BGM鑑賞

## 対象ユーザー

- ノベルゲームを作りたい人 — ブロックを並べるだけで演出・分岐まで組める
- ノベルゲームを遊びたい人 — インストール不要、ブラウザで即プレイ
