## 説明文

ブラウザだけでビジュアルノベルを制作・公開・プレイできるプラットフォーム「kaedevn（かえで）」の操作デモ動画です。

▼ この動画の内容
0:00 ログイン → マイページ
0:15 ブロックエディタを開く
0:40 背景ブロック追加 → 公式ファンタジー素材を選択
1:10 テキストブロック追加 → セリフ入力
1:40 全ブロックのプロパティ確認
2:15 キャラクター拡大・位置調整（スケール3倍、Y座標1400）
2:45 プレビュー実行（タイトル画面 → シナリオ再生）

▼ 主な機能
- ブロック（GUI）とスクリプト（テキスト）が双方向同期するハイブリッド・エディタ
- 13種類のブロック（背景・キャラ・テキスト・選択肢・BGM・SE・アニメーション等）
- タイムラインアニメーション、15種のスクリーンフィルター（GLSL）
- ファンタジー・現代学園の背景・立ち絵・BGM・SEを標準搭載
- ブラウザで即プレイ、PWA対応、スマホタッチジェスチャー対応

▼ 技術スタック
- エディタ: React + Zustand + Vite
- エンジン: PixiJS (WebGL) + Web Audio API
- バックエンド: Hono + PostgreSQL + Azure
- ネイティブエンジン: SDL2 + C++（Switch対応予定）

▼ AI開発
この動画の撮影・編集はClaude Opus 4.6がPlaywright E2Eテストで自動実行しています。
