## 1. エディタ機能一覧

### ブロックエディタ（14種類）

| カテゴリ | ブロック | 内容 |
|---------|---------|------|
| 基本 | start / text / bg / ch / overlay | ページ開始、セリフ、背景、キャラクター、オーバーレイ |
| 制御 | choice / if / set_var / jump | 選択肢分岐、条件分岐、変数設定、ページジャンプ |
| 演出 | effect / screen_filter / timeline | 画面効果（8種）、フィルター（15種）、キーフレームアニメ |
| 拡張 | battle / ksc | ターン制RPGバトル、スクリプト直書き |

### スクリプト言語

| | KS（Kaede Script） | KSC（Kaede Script Code） |
|---|---|---|
| 難易度 | タグを覚えるだけ | TypeScript風コード |
| コマンド数 | 20 | HostAPI メソッド群 |
| ループ / 関数 | 不可 | for / while / def / sub |
| 型チェック | なし | あり |
| デバッグ | — | ブレークポイント、変数ウォッチ、ステップ実行 |
| エディタ | ブロックと双方向同期 | Monaco Editor（シンタックスハイライト、補完、リアルタイム診断）|

> 制作モードの詳細は別紙「制作モードガイド」を参照

### 演出機能

**スクリーンフィルター（15種・GLSLシェーダー）**

Bloom / Chromatic Aberration / CRT / GameBoy / Glitch / Night / Pixelate / Rain / Underwater / Vignette / PC-98 / ColorTint / FocusBlur / Noise / OldFilm

**タイムラインアニメーション**
- 4トラック構成: カメラ / エンティティ / オーディオ / イベント
- キーフレーム + イージング関数
- カメラ: パン / ズーム / シェイク

**バトルシステム**
- ターン制コマンドRPG
- HP / MP / ATK / DEF / SPD
- 攻撃スキル、回復、バフ / デバフ、状態異常（毒 / 麻痺 / スタン）
- AI敵、シード付き決定論的結果
- 勝敗でストーリー分岐

### エディタ UX

- 自動保存（60秒間隔）+ 未保存離脱警告
- 50段階 Undo / Redo
- ブロック複製（Ctrl+D）
- ドラッグ&ドロップ並べ替え
- 未設定ブロックの視覚警告（オレンジ枠）
- レスポンシブ: デスクトップ3カラム / タブレット2カラム / モバイル1カラム

## 2. プレイヤー機能一覧

### 基本

- タイプライター表示（速度可変）、話者名表示
- 選択肢分岐（条件付き表示対応）
- 「つづきから」（オートセーブ / クイックセーブ検出）

### セーブ / ロード

- 手動20スロット（サムネイル + ページ切替UI）
- クイックセーブ / ロード（F5 / F9）
- オートセーブ（60秒間隔 + beforeunload）
- クラウドセーブ（アカウント連携・複数端末対応、API実装済み）

### モード

- オートモード / スキップ（既読スキップ対応）
- NVLモード（全画面テキスト）/ ADVモード切替
- ウィンドウ非表示（背景鑑賞）

### スマホ最適化

- タッチジェスチャー: スワイプ上→バックログ、スワイプ下→メニュー、2本指タップ→メニュー
- レスポンシブUI: dvh + safe-area-inset 対応
- テキストサイズ（18〜32px）/ フォント変更（ゴシック / 明朝）

### おまけ・コレクション

- ギャラリー: 背景CG / キャラ立ち絵 / イベントCG（カテゴリタブ、達成率、未解放「?」表示）
- BGMギャラリー: 試聴 / 停止 / ページ送り
- チャプターセレクト（エンジン内UI）
- エンドリスト（到達エンディング一覧・達成率）

### アクセシビリティ

- 色覚サポート: P型 / D型 / T型 + 高コントラストモード
- Web Share API: スクリーンショット + テキストを SNS にワンタップ共有
- PWA: ホーム画面追加でアプリ同等体験

### キーボードショートカット（PC）

`S`=スキップ `A`=オート `L`=バックログ `M`=メニュー `F5`=クイックセーブ `F9`=クイックロード `H`=UI非表示 `P`=スクリーンショット

## 3. AI執筆アシスタント

> 現在テスト運用中。正式リリース時に一般提供予定。

### 4段階パイプライン

1. **章プロット生成** — 物語の大枠を生成
2. **話プロット分割** — 章をシーン単位に分割
3. **セリフ・地の文執筆** — シーンごとにテキスト生成
4. **スクリプト変換** — テキストをKS/KSCスクリプトに変換（テンプレート処理、LLM不使用）

### 対応プロバイダ

- OpenAI（GPT-4o / GPT-4o-mini）
- Anthropic（Claude Sonnet / Opus）
- Google AI（Gemini）

### CLI ツール

- 世界設定・キャラクター設定の .md ファイルをインプットに、複数話を一括生成
- 7ジャンル（ファンタジー、学園、SF、ミステリー等）のプロジェクト一括生成に対応

## 4. ネイティブエンジン（SDL2 + C++）

### コード構成（約2,300行）

| コンポーネント | 行数 | 役割 |
|--------------|------|------|
| SDL2Engine | ~400 | 描画・演出・UI |
| Interpreter + Parser + Evaluator | ~620 | KSスクリプト実行 |
| Shader | ~100 | GLSLシェーダー管理 |
| AssetProvider + AudioManager | ~160 | アセット読み込み・音声再生 |
| FileStorage + TextureManager + etc. | ~160 | セーブ/ロード・テクスチャ・フォント |
| テストコード | ~280 | ユニットテスト6ファイル |

### 動作確認済みプラットフォーム

- Android（SDL2 Activity）
- iOS（Safe Area対応）
- macOS / Linux

### Switch 移植見積もり

追加コード: 約1,500〜2,000行（プラットフォーム初期化、Joy-Con入力、Nintendo eShop連携等）
