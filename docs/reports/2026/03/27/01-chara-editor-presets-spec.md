## プリセットUI構成

```
[Female] [Male]          ← 性別タブ（上位）

Presets                  ← 性別に応じて4人が表示
┌─────────────────────────────────────┐
│ [花子] [美咲] [さくら] [静江]       │  ← Female 選択時
│ [太郎] [健一] [翔太]   [正雄]       │  ← Male 選択時
└─────────────────────────────────────┘
```

- 性別タブ切り替えで表示されるプリセットが変わる
- プリセット選択で体型・髪・色が一括セット
- 表情・ポーズ・リップシンクは選択後に個別変更

## 新規作成が必要なプリセット

| キャラ | 作業内容 |
|--------|---------|
| 静江（女性Elder） | getDefaults("female") ベースで年配女性パラメータを新規定義 |
| 翔太（男性少年） | getDefaults("male") ベースで少年パラメータを新規定義 |

既存の6プリセットのうち4つ（Girl Standard, Tall Woman, Child, Elder）はそのままリネームして使用。
Muscular Man, Boy Standard もそのままリネーム。
