# リップシンク機能 仕様・実装リファレンス

> Generated by Claude Opus 4.6

## モード一覧

| モード | 動作 |
|--------|------|
| **OFF** | 表情の口をそのまま使う |
| **Talk** | 通常会話の口パク（8fps） |
| **Whisper** | ささやき・小さい口の動き（5fps） |
| **Shout** | 叫び・大きく開く口（10fps） |

## 母音と口形状

日本語の全文字を 6 種類の口形状にマッピングする。

| 母音 | 口の形 | 対応する文字 |
|------|--------|------------|
| **A** | 大きく開く | あ行・か行・さ行…の「あ段」 |
| **I** | 横に広い | い段 |
| **U** | すぼめる | う段 |
| **E** | やや開く | え段 |
| **O** | 縦に開く | お段 |
| **N** | 閉じる | ん・っ・句読点・記号 |

### 特殊文字の扱い

| 文字 | 処理 |
|------|------|
| `ー`（長音） | 直前の母音を維持 |
| `。` `、` `！` `？` `…` | N（閉じる） |
| 漢字・英字 | A（口を開ける） |
| スペース | N（閉じる） |

## モード別口の開き具合

同じ母音でもモードによって開き具合が変わる。

| 母音 | Talk（×1.0） | Whisper（×0.4） | Shout（×1.5） |
|------|-------------|----------------|--------------|
| A の開き | 0.8 | 0.32 | 1.2 |
| I の横幅 | 1.4 | 1.16 | 1.6 |
| U の開き | 0.4 | 0.16 | 0.6 |
| O の開き | 0.6 | 0.24 | 0.9 |

## 動作モード

### テキストありの場合

テキストを 1 文字ずつ表示しながら、文字→母音→口形状を決定する。

```
「こんにちは」
  ↓ 文字→母音変換
  O → N → N → I → A
  ↓ モード別強度で口形状を生成
  (Talk) O(普通) → N(閉) → N(閉) → I(普通) → A(普通)
  (Shout) O!(大) → N(閉) → N(閉) → I!(大) → A!(大)
```

表示速度はモードで変わる:
- Talk: 80ms/文字
- Whisper: 120ms/文字
- Shout: 50ms/文字

### テキストなしの場合

あらかじめ定義されたパターン配列を循環再生する。

```typescript
talk:    ["N","A","I","N","U","E","N","O","A","N","I","N","A","E","N","O","N","A","N","N"]
whisper: ["N","A","N","I","N","N","U","N","N","E","N","N","A","N","N","O","N","N","N","N"]
shout:   ["A","A","O","A","E","A","O","A","A","I","A","O","A","A","E","O","A","A","O","A"]
```

## テキスト内のモード切り替え（設計済み・未実装）

1 つのセリフの中でモードを切り替えるためのインラインタグ。

```
[w]ごめんね...[s]やめてーー！！
```

| タグ | モード |
|------|--------|
| `[t]` | talk（デフォルト、省略可） |
| `[w]` | whisper |
| `[s]` | shout |

### ルール

- テキスト先頭にタグがない場合は `[t]`（talk）として扱う
- タグは表示時には画面に出さない
- 連続する同じタグは無視（`[t][t]` → `[t]`）

### エディタ UI（設計済み・未実装）

```
┌──────────────────────────────────┐
│ [t]こんにちは、[w]ごめんね...[s] │
│ やめてーー！！                    │
└──────────────────────────────────┘
[ ▼ Talk / Whisper / Shout ] [挿入]
```

- テキストエリア内でカーソルを置く
- プルダウンからモードを選ぶ（`selectionStart` でカーソル位置を保存）
- 「挿入」ボタンでタグを挿入
- カーソルはタグの直後に移動

### モードの設定者

| 設定者 | 方法 |
|--------|------|
| **作者** | エディタ UI でタグを挿入 |
| **Claude Code** | シナリオ自動生成時にタグを付与 |

### 主要な関数・定数

| 名前 | 場所 | 役割 |
|------|------|------|
| `LIP_SHAPES` | chara-editor.ts | 母音 6 種の口形状定義（mouthCurve, mouthOpen, mouthWidth） |
| `MODE_INTENSITY` | chara-editor.ts | モード別強度倍率（talk=1.0, whisper=0.4, shout=1.5） |
| `KANA_VOWEL_MAP` | chara-editor.ts | かな→母音の変換テーブル |
| `charToVowel(ch)` | chara-editor.ts | 1 文字→母音を返す |
| `textToVowels(text)` | chara-editor.ts | テキスト→母音列を生成 |
| `getModedLipShape(vowel, mode)` | chara-editor.ts | 母音+モード→口形状を返す |
| `getActiveLipMode()` | chara-editor.ts | 現在選択中のモードを返す |
| `playTextLip(text, mode?)` | chara-editor.ts | テキスト再生を開始 |
| `TALK_PATTERNS` | chara-editor.ts | テキストなし時のパターン配列 |
