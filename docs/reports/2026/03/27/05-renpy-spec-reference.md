# Ren'Py ビジュアルノベルエンジン 仕様リファレンス

> Generated by Claude Opus 4.6

kaedevn との競合比較のため、Ren'Py の主要機能をまとめる。

## 1. スクリプト言語

Python ベースの独自言語（`.rpy` ファイル）。

```renpy
label start:
    "ナレーション"
    e "こんにちは！"
    e happy "嬉しいな！"       # 表情指定

menu:
    "選択肢A":
        jump route_a
    "選択肢B" if has_item:     # 条件付き
        jump route_b

if affection >= 10:
    e "大好き！"

default affection = 0          # セーブ対象変数
define e = Character("えみり") # 定数（セーブ対象外）
```

## 2. キャラクター・表情差分

### 基本定義

```renpy
define e = Character("えみり", color="#c8ffc8", image="eileen")
```

### Layered Image（レイヤード画像）

複数レイヤー（表情・衣装・アクセサリ）を独立制御。

```renpy
layeredimage eileen:
    always:
        "eileen_base"
    group expression:
        attribute happy default:
            "eileen_happy"
        attribute sad:
            "eileen_sad"
        attribute angry:
            "eileen_angry"
    group outfit:
        attribute uniform default:
            "eileen_uniform"
        attribute casual:
            "eileen_casual"
    if is_night:
        "eileen_night_overlay"
```

```renpy
show eileen happy casual     # 表情 + 衣装の組み合わせ
show eileen sad              # 衣装そのまま、表情だけ変更
```

## 3. テキスト内タグ（インラインタグ）

### スタイリング系

| タグ | 用途 | 例 |
|------|------|-----|
| `{b}...{/b}` | 太字 | `{b}重要{/b}` |
| `{i}...{/i}` | 斜体 | `{i}ささやき{/i}` |
| `{u}...{/u}` | 下線 | |
| `{s}...{/s}` | 取り消し線 | |
| `{color=#hex}...{/color}` | 文字色 | `{color=#ff0000}赤{/color}` |
| `{size=N}...{/size}` | フォントサイズ | `{size=+10}大きい{/size}` |
| `{font=name}...{/font}` | フォント指定 | |
| `{alpha=N}...{/alpha}` | 透明度 | `{alpha=0.5}半透明{/alpha}` |
| `{outlinecolor=#hex}` | アウトライン色 | |

### 制御系

| タグ | 用途 | 例 |
|------|------|-----|
| `{w}` | クリック待ち | `前半{w}後半` |
| `{p}` | 改ページ待ち | `前半{p}次ページ` |
| `{nw}` | 待ちなし（即次へ） | `自動進行{nw}` |
| `{fast}` | 即時表示 | `{fast}ここから表示` |
| `{cps=N}` | 文字表示速度(文字/秒) | `{cps=5}ゆっくり{/cps}` |
| `{cps=*N}` | 速度倍率 | `{cps=*2}速い{/cps}` |
| `{space=N}` | スペース挿入 | |
| `{vspace=N}` | 垂直スペース | |

### その他

| タグ | 用途 |
|------|------|
| `{a=URL}...{/a}` | ハイパーリンク |
| `{image=path}` | インライン画像 |
| `{rb}...{/rb}` + `{rt}...{/rt}` | ルビ（振り仮名） |

### カスタムテキストタグ

```renpy
python early:
    def my_tag(tag, argument, contents):
        return [(renpy.TEXT_TEXT, "加工後テキスト")]
    config.custom_text_tags["mytag"] = my_tag
```

### kaedevn との比較

| 機能 | Ren'Py | kaedevn（提案） |
|------|--------|----------------|
| テキストスタイル | `{b}`, `{color}` 等 | 未実装 |
| 文字速度 | `{cps=N}` | 未実装 |
| クリック待ち | `{w}`, `{p}` | `[wait:ms]` |
| **表情変更** | **なし（別コマンド）** | **`[e:smile]`** |
| **ポーズ変更** | **なし（別コマンド）** | **`[p:wave]`** |
| **リップシンク** | **なし** | **`[t]`, `[w]`, `[s]`** |
| **カメラ演出** | **なし（別コマンド）** | **`[cam:shake]`** |

**Ren'Py はテキスト内での表情・ポーズ変更をサポートしていない。** スタイリングと制御タグのみ。表情変更は `show` コマンドとして別行に書く必要がある。ここが kaedevn の差別化ポイント。

## 4. 画面演出

### トランジション

```renpy
show eileen happy with dissolve
scene bg room with fade
with Dissolve(1.0)              # 時間指定
with Fade(0.5, 0.5, 0.5)       # out, hold, in
with pixellate / blinds / squares / wipeleft
with vpunch / hpunch            # 画面振動
```

### ATL（Animation and Transformation Language）

Ren'Py 最大の強み。宣言的にアニメーション定義。

```renpy
transform my_animation:
    xalign 0.5 yalign 1.0
    alpha 0.0
    linear 1.0 alpha 1.0        # 1秒フェードイン
    pause 0.5
    ease 0.5 xalign 0.8         # イージング移動
    repeat                       # ループ

show eileen at my_animation
```

ATL プロパティ: `pos`, `anchor`, `align`, `zoom`, `rotate`, `alpha`, `crop`, `offset`
補間関数: `linear`, `ease`, `easein`, `easeout`, `pause`

### カメラ制御

```renpy
camera:
    perspective True
    xpos 100 ypos 50
    linear 1.0 xpos 200          # カメラパン

show layer master:
    zoom 1.5
    linear 0.5 zoom 1.0          # ズームアウト
```

### 画像フィルター（matrixcolor）

```renpy
show eileen at Transform(matrixcolor=SaturationMatrix(0.0))  # グレースケール
show eileen at Transform(matrixcolor=BrightnessMatrix(-0.3))  # 暗く
show eileen at Transform(matrixcolor=TintMatrix("#aaaaff"))   # 青みがかり
```

## 5. 音声・BGM・SE

3チャンネル: `music`（BGM）, `sound`（SE）, `voice`

```renpy
play music "bgm/theme.ogg" fadein 1.0
play music ["bgm/intro.ogg", "bgm/loop.ogg"]  # イントロ+ループ
stop music fadeout 1.0

play sound "se/click.ogg"
play sound "se/rain.ogg" loop

voice "voice/ch01_001.ogg"
e "ボイス付きセリフ"

# ボイス自動化
define config.auto_voice = "voice/{id}.ogg"
```

対応フォーマット: Ogg Vorbis（推奨）, MP3, WAV, Opus

## 6. セーブ/ロード

- **自動**: `default` 変数、画面状態、BGM が自動保存・復元
- **ロールバック**: 標準搭載（マウスホイールで巻き戻し）
- **永続データ**: `persistent.*` でセーブスロット横断の共有データ
- **セーブスロット**: `auto-N`, `quick-N`, ユーザースロット

```renpy
default persistent.endings_seen = set()
$ persistent.endings_seen.add("true_end")
```

## 7. UI カスタマイズ（Screen Language）

```renpy
screen say(who, what):
    window:
        vbox:
            if who:
                text who style "say_label"
            text what style "say_dialogue"

screen main_menu():
    tag menu
    add "gui/main_menu.png"
    vbox:
        textbutton "スタート" action Start()
        textbutton "ロード" action ShowMenu("load")
        textbutton "設定" action ShowMenu("preferences")
        textbutton "終了" action Quit()
```

`gui.rpy` でグローバル設定（フォント、サイズ、カラー等）。ADV / NVL 2モード。

## 8. Live2D 対応

Ren'Py 7.4+ で公式サポート。

```renpy
image eileen = Live2D("live2d/eileen", base=0.5)
show eileen m1    # モーション
show eileen e2    # 表情
```

制限: Cubism SDK の別途ライセンスが必要（商用時）。

## 9. マルチプラットフォーム

| プラットフォーム | 対応 |
|----------------|------|
| Windows / macOS / Linux | 公式サポート |
| Android / iOS | 公式サポート |
| Web (HTML5) | 公式サポート（8.x で強化） |
| **Nintendo Switch** | **サードパーティ移植のみ**（公式ビルド不可） |
| PlayStation / Xbox | 同上 |

**Switch 対応は Ren'Py 最大の弱点。** サードパーティ（Ratalaika Games 等）が個別に移植する形で、公式のビルドパイプラインは存在しない。

## 10. kaedevn の差別化ポイント

| 領域 | Ren'Py の弱み | kaedevn の強み |
|------|-------------|---------------|
| **Switch 対応** | サードパーティ移植のみ | ネイティブ対応（設計目標） |
| **インライン演出** | テキスト内で表情・ポーズ変更不可 | `[e:][p:][cam:]` で自在に制御 |
| **リップシンク** | 標準機能なし | テキスト駆動リップシンク搭載 |
| **図形キャラ** | なし（画像必須） | 画像なしで動くキャラクター |
| **エディタ** | テキストエディタのみ | ブロックエディタ + KSC エディタ |
| **AI 連携** | なし | Claude Code でシナリオ自動生成 |

### Ren'Py が勝っている領域

| 領域 | 詳細 |
|------|------|
| **ATL** | 宣言的アニメーション言語は非常に表現力が高い |
| **Screen Language** | UI を柔軟にカスタマイズできる |
| **エコシステム** | 巨大なコミュニティ、豊富なチュートリアル・アセット |
| **実績** | Steam 上の多数の商用タイトル |
| **ロールバック** | 標準で巻き戻し機能 |
| **Layered Image** | 表情 × 衣装 × アクセサリの組み合わせ管理 |
