# Nintendo Switch リリースへの思い

## ゲーム業界への挑戦

若い頃、ゲーム会社に転職したいと思っていましたが、2次面接まで進んで落ちました。
その後も組み込み系の会社で働きながら「ゲームの仕事がしたい」と思い続け、やがてスマホゲーム開発を仕事にできるようになりました。

一時は Android・iOS でカジュアルゲームを小規模なチーム（個人事業主の集まり）でリリースしていましたが、継続を断念し、今は普通の IT 技術者として働いています。

## 自作ゲームへの道

その頃から「自分でゲームをリリースしよう」と Unity を使って試作を繰り返していましたが、いまだリリースには至りませんでした。

ChatGPT と壁打ちするうちに「TypeScript とブラウザはどうだろう？」という発想に至り、Claude Code と一緒に作ってみたら、思いの外、ノベルゲームがサクッと完成しました。

## プラットフォームという選択

自分にはゲームを作るスキルがなかったので、「ノベルゲームを作る・遊ぶができるプラットフォーム」を作ることにしました。そうしたら、1ヶ月でベータ版リリースまで到達できました。

また、ラノベやなろう、カクヨムはよく読んでいて、1000作品以上は読んだと思います。アニメ化されて無料で読めるものはほぼ全部読んでいますし、気に入ったものは購入もしました。

そんな経緯から、「ノベルゲーム版の『なろう』『カクヨム』を作ってもいいのでは」と思い、このプラットフォームを作りました。

## まとめ

1. 自分にはゲームを作るスキルがないので、プラットフォームを作った
2. なろう・カクヨムのように、ノベルゲームが簡単に作れて Switch に出せる世界を目指した
3. ゲームとしての最終形態「Switch でリリース」を、最初からとても意識して設計した

以上、開発者の思いをお伝えしました。
