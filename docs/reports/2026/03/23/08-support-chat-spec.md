# サポートチャット仕様書 — Discord 埋め込み方式

> Generated by Claude Opus 4.6

## 方針

自前のチャット機能は作らない。**Discord をサポートチャネルにする。**

理由:
- やりとりがオープンになり、他の作者も参考にできる
- クローズドなサポートチャットでは、せっかくの Q&A が他の人に見えない
- 作者同士のコミュニティ形成にもつながる
- Discord のインフラ（通知・モバイル・検索）をそのまま使える
- 開発・運用コストゼロ

## 実装内容

### エディタ・Next.js の各画面に Discord リンクを常設

| 設置箇所 | 方法 |
|---------|------|
| エディタ（apps/editor）ヘッダー | 「💬 質問する」ボタン |
| マイページ（apps/next）サイドバー/ヘッダー | 「💬 Discord で質問」リンク |
| プロジェクト詳細（apps/next） | フッターに「困ったら Discord で質問」 |
| ランディングページ（apps/next） | フッターに Discord リンク |

### リンク先

```
https://discord.gg/{INVITE_CODE}
```

環境変数 `NEXT_PUBLIC_DISCORD_URL` / `VITE_DISCORD_URL` で管理。未設定時はリンクを非表示。

### チャネル構成（Discord 側）

| チャネル | 用途 |
|---------|------|
| #質問-相談 | エディタの使い方、バグ報告 |
| #素材リクエスト | 「こんな背景がほしい」→ 運営が対応 |
| #作品紹介 | 自作の宣伝、感想交換 |
| #雑談 | 自由トーク |
| #お知らせ | 運営からの通知（read-only） |

### 画像リクエストの流れ

```
作者が #素材リクエスト に投稿
  → 「学園の屋上の背景がほしいです」
  → 運営が確認 → AI で生成 → 公式アセットに追加
  → 「追加しました！」と返信
  → 他の作者も「いいね、私も使いたい」
```

クローズドチャットでは生まれない、**リクエストの共有と発見** が起きる。
