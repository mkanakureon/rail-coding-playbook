> Generated by Claude Opus 4.6

## 目的

作者が「最後まで読まれているか分からない」「誰にも読まれない」という2つの恐怖を解消し、100人の作者を定着させる。

## データモデル

### WorkPageStats（新規）

```prisma
model WorkPageStats {
  id        String @id @db.VarChar(31)
  workId    String @map("work_id") @db.VarChar(31)
  pageIndex Int    @map("page_index")
  viewCount Int    @default(0) @map("view_count")
  totalMs   BigInt @default(0) @map("total_ms")  // 滞在時間の合計

  work      Work   @relation(fields: [workId], references: [id], onDelete: Cascade)

  @@unique([workId, pageIndex])
  @@index([workId])
  @@map("work_page_stats")
}
```

### UserBadge（新規）

```prisma
model UserBadge {
  id        String @id @db.VarChar(31)
  userId    String @map("user_id") @db.VarChar(31)
  badgeType String @map("badge_type") @db.VarChar(50)  // "words_1000", "scenes_5", "first_publish"
  earnedAt  BigInt @map("earned_at")

  user      User   @relation(fields: [userId], references: [id], onDelete: Cascade)

  @@unique([userId, badgeType])
  @@index([userId])
  @@map("user_badges")
}
```

## API エンドポイント

### アナリティクス

| メソッド | パス | 説明 | 認証 |
|---------|------|------|------|
| POST | `/api/analytics/page-view` | ページ閲覧ビーコン（workId, pageIndex, durationMs） | 不要（公開作品） |
| GET | `/api/projects/:id/analytics` | 離脱ファネル・読了率・滞在時間 | authMiddleware |

### 新着ブースト

| メソッド | パス | 説明 | 認証 |
|---------|------|------|------|
| GET | `/api/works?sort=fresh` | フレッシュ・ピックアップ（playCount < 50 を優先） | 不要 |

### バッジ

| メソッド | パス | 説明 | 認証 |
|---------|------|------|------|
| GET | `/api/users/me/badges` | 自分のバッジ一覧 | authMiddleware |
| POST | `/api/users/me/badges/check` | バッジ達成チェック（エディタから呼ぶ） | authMiddleware |

### テンプレート

| メソッド | パス | 説明 | 認証 |
|---------|------|------|------|
| GET | `/api/templates` | テンプレート一覧 | 不要 |
| POST | `/api/projects/from-template` | テンプレートからプロジェクト作成 | authMiddleware |

## ビーコンの負荷対策

- **レート制限**: 同一セッション + 同一ページで60秒に1回まで
- **非同期書き込み**: `await` せず fire-and-forget で送信
- **開発環境無効化**: `NODE_ENV !== 'production'` のときは無視
- **将来**: Redis でバッファリング → バッチ書き込み（MAU 5,000 以降）
