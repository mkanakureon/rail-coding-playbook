## 3. アクティブ作者 500 の定義と内訳

### 定義

| セグメント | 人数 | 行動パターン |
|-----------|-----:|------------|
| ヘビー（週3回以上） | 50 | 連載中。毎週更新。動画も定期投稿 |
| レギュラー（週1回） | 150 | 週末に制作。月1〜2回公開 |
| ライト（月1〜2回） | 200 | テンプレートをカスタマイズ。たまに更新 |
| 復帰（休眠→再開） | 100 | フレッシュピックアップやリアクション通知で戻ってきた |
| **合計** | **500** | |

### アクティブ率

- 登録1,200人 中 500人がアクティブ = **41.7%**
- 一般的な UGC プラットフォームのアクティブ率（20〜30%）より高い想定
- 高い理由: フレッシュピックアップ + リアクション + Discord の3重の継続施策

### 達成条件

- フレッシュ・ピックアップで新作が48時間以内に10プレイ獲得
- リアクション（❤️😂😭⚡️）が作品公開後24時間以内に届く
- Discord で週1回以上の投稿がある（制作祭・雑談）
- 読了アナリティクスで「どこで離脱されたか」が見える

## 6. 営業利益 27万円の意味

| 指標 | 値 |

| 売上 | 55万 |

| 営業利益率 | **67%** |

### M3 で37万円が意味すること — 本業移行の判断ライン

現在の給与: **月30万円**

| 指標 | 給与（現職） | M3 営業利益 |
|------|:-----------:|:----------:|
| 月額 | 30万（確定） | 37万（予測） |
| 社会保険 | 会社負担あり | 自己負担（国保+年金 約6万） |
| 実質手取り | ~24万 | ~31万 |
| M4 以降 | 30万のまま | 67万 → 102万 → 138万 |

**M3 時点で給与を超える。** M4 以降は2倍以上に拡大。

#### リスク評価

| リスク | 対策 |
|--------|------|
| MAU が予測に届かない | 保守ケース（MAU 25,000）でも売上30万前後。インフラ18万を差し引いて12万。最低ラインは赤字にならない |
| 課金率が低い | 作者課金（容量+書き出し）が収益の柱。読者課金がゼロでも作者課金25万で黒字 |
| M3 に届かず M5 にずれる | M5 で102万。3ヶ月の遅延は貯蓄で吸収可能な範囲か確認 |

#### 判断基準

- **M2 終了時に MAU 15,000 以上 + 登録作者 400 以上** → M3 達成見込みあり → 本業移行 GO
- **M2 終了時に MAU 10,000 未満** → M3 が半年ずれる可能性 → 副業継続

#### 必要な貯蓄バッファ

- 最悪ケース（M3 が6ヶ月遅延）: 生活費25万 × 6ヶ月 = **150万円**
- 推奨: **200万円** の貯蓄があれば安全に移行可能

### その他

- **黒字化が定着**: M2 で単月黒字化（13万）→ M3 で37万に拡大
- **固定費が低い**: インフラ12万 + その他3万 = 15万。人件費0で利益率67%
- **成長投資の原資**: 37万の利益から月10〜15万を広告・プロモーションに回せる
- **M6 まで1人で回せる**: 問い合わせ・画像リクエスト・Discord は自分で対応

### M4 への展望

M3 の27万を足場に:
- 広告出稿開始（月5〜10万）→ MAU を 50,000 → 100,000 に加速
- 動画投稿の本数が増え、SNS 流入が自律成長
- Switch 2本目以降の申請
- M4 目標: **売上90万、営業利益52万**
