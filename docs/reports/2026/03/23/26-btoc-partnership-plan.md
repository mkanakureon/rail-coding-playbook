## 出口戦略: kaedevn の引き継ぎ先としての Minto

### 背景

kaedevn 創業者は61歳（2026年時点）。事業が成長した後、働けなくなる前に kaedevn を引き継ぐ必要がある。Minto は引き継ぎ先として最適。

### なぜ Minto か

| 引き継ぎ先の条件 | Minto の適格性 |
|---------------|-------------|
| IP ビジネスを理解してる | 本業（スタンプ60億DL、ライセンス700社） |
| ゲームプラットフォーム運営できる | FANY:D 運営中、CryptoCrystal 運営実績 |
| アセットの価値がわかる | スタンプ/キャラクター IP が本業 |
| 社長と信頼関係がある | 元同僚 |
| 会社の規模が十分 | 141人、14.6億調達 |
| 海外展開できる | 上海・バンコク・ホーチミンに拠点 |
| 技術チームがある | エンジニアが在籍。引き継ぎ可能 |

### タイムライン

```
M6:     パートナーとして連携開始
          → Minto が kaedevn のエンジン・運営を理解する
          → 一緒にゲームを出す
          → 信頼関係が深まる

M24-M36: Minto が kaedevn の中身を十分理解した段階
          → 引き継ぎの相談を開始
          → 形式（事業譲渡 or M&A）を決定

引き継ぎ:  技術顧問として一定期間関与し、完全移行
```

### M6 のパートナーシップが引き継ぎの伏線になる

M6 から一緒にゲームを出すことで、Minto は kaedevn のエンジン・インフラ・アセットライブラリ・作者コミュニティを内側から理解する。引き継ぎ時に「知らないものを渡される」状態にならない。

### 引き継ぎの形式

| 形式 | 内容 | kaedevn 側のメリット |
|------|------|-------------------|
| 事業譲渡 | kaedevn の事業を Minto に売却 | 一括で対価を受け取れる |
| 吸収合併 | Minto が kaedevn を吸収 | Minto 株式での対価もあり得る |
| 技術顧問として残る | 引き継ぎ期間中は顧問として関与 | 段階的な引退が可能 |

### kaedevn の事業価値（M36 時点の推定）

| 資産 | 推定価値 |
|------|---------|
| 年間営業利益 3.2億 × 倍率 5-10倍 | **16-32億** |
| アセットライブラリ ~40,000点 | 再構築コスト5億以上 |
| 作者コミュニティ 23,800人 | 代替不可能 |
| MAU 241万 | ユーザー基盤 |
| 運営型ゲームアプリ基盤 + パートナー網 | Minto の既存事業と相乗効果 |
| Switch 対応設計 | コンソール展開の可能性 |

### Minto にとっての引き継ぎメリット

| メリット | 内容 |
|---------|------|
| ゲーム事業の強化 | CryptoCrystal / The Sandbox の次の柱 |
| アセットライブラリ取得 | 40,000点の2D素材。自社 IP のゲーム化に即活用 |
| 作者コミュニティ | 23,800人の作者 = コンテンツ生成力 |
| 技術基盤 | ゲームエンジン + インフラ + ガチャ/課金基盤を一括取得 |
| 海外展開の基盤 | ローカライズ済みなら Minto の海外拠点で展開可能 |

### 重要: 早めに関係を作る

引き継ぎは突然できない。M6 からパートナーとして一緒に動き、Minto が kaedevn の価値を理解した上で、自然な流れで引き継ぎの話ができる状態を作る。**働けなくなってから慌てるのではなく、元気なうちに準備を完了する。**
