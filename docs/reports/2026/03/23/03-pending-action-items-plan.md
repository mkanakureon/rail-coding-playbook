## A-1. Health Dashboard

管理画面に `/api/admin/health-dashboard` エンドポイントを追加。

返すデータ:
- DB 接続プール状態（接続数/最大数）
- メモリ使用量（heapUsed/heapTotal/rss）
- プロセス稼働時間
- 直近のエラー数（ログから集計）
