# アセットチーム体制計画

> Generated by Claude Opus 4.6

> 更新日: 2026-03-23

## 概要

アセット発注を夫婦2人（ゲーム経験者、現職ソシャゲ運用）に全委任する。制作・外注管理・品質管理を全て夫婦経由にし、自分のアセット管理工数をゼロにする。**全期間を通じて業務委託。正社員雇用はしない。**

---

## 体制

```
自分 ──「来月はSFジャンル強化で」──→ 夫婦（2人・業務委託）──→ 外注絵師A
                                        │                    ──→ 外注絵師B
                                        │                    ──→ 外注音楽C
                                        │                    ──→ ...
                                        └─ 自分たちでも制作
```

### 夫婦の役割

| 役割 | 内容 |
|------|------|
| 自主制作 | ドット絵、マップチップ、キャラチップ等を自分たちで制作 |
| 外注管理 | 外注先の発掘、発注、仕様書作成、納品チェック、リテイク指示 |
| 品質管理 | スタイルガイドの策定・維持、全アセットの品質統一 |
| 予算管理 | アセット予算内での配分（自主制作 vs 外注） |
| ディレクション | ジャンル別の優先順位、不足素材の特定 |

### 自分の関与

| 項目 | 内容 |
|------|------|
| 月1回 | 方針指示（「次はホラージャンル」「キャラチップ増やして」等） |
| 利用データ共有 | どのアセットが使われているかのデータを渡す |
| 予算承認 | アセット予算額の確定（利益の20%ルール） |
| **それ以外** | **なし** |

## なぜ業務委託のままか

### 正社員 vs 業務委託

| 項目 | 正社員雇用 | 業務委託（発注） |
|------|-----------|---------------|
| 社会保険 | 会社負担15%（2人で月14万） | **なし** |
| 雇用リスク | 解雇しにくい | **いつでも調整可能** |
| 夫婦の本業 | 辞める必要がある | **辞めなくていい。副業→自分の判断で専業に** |
| 月額コスト | 90万（給与+社保） | **予算全額を制作費に使える** |
| 3年間の社保差額 | 約420万（14万×30ヶ月） | **¥0（全額がアセットに回る）** |

### 夫婦にとってのメリット

| メリット | 内容 |
|---------|------|
| 本業を辞めなくていい | 現職（ソシャゲ運用）を続けながら副業で開始 |
| リスクゼロ | kaedevn の発注が減っても本業がある |
| 自分のペースで判断 | 発注が安定して「これでやっていける」と思ったら専業に転向 |
| 「辞めてくれ」がない | 業務委託だから雇用関係の縛りがない |

### kaedevn にとってのメリット

| メリット | 内容 |
|---------|------|
| 社保コスト ¥0 | 3年で420万浮く → その分アセットが増える |
| 柔軟性 | 発注量を月ごとに調整可能 |
| 利益率維持 | 人件費が増えないので77%の利益率を維持 |
| 固定費が増えない | 売上が下がったら発注を減らせばいい |

## 発注計画

### 全期間業務委託

| 時期 | アセット予算/月 | 夫婦の稼働 | 夫婦への発注額/月 |
|------|------------:|----------|---------------:|
| M6-M11 | 30-133万 | 副業 | 30-133万（全額） |
| M12-M17 | 158-319万 | 副業 or 専業（夫婦の判断） | 158-319万（全額） |
| M18以降 | 349万〜 | 専業の可能性高い | 349万〜（全額） |

**予算は全額が夫婦への発注。** 夫婦が自主制作するか、さらに外注に回すかは夫婦の判断。

### M6 から業務委託で始める理由

| メリット | 内容 |
|---------|------|
| テイスト統一 | M6 から同じ2人が作るので、最初からスタイルが一定 |
| 品質基準が固まる | 「kaedevn の素材はこういうもの」が早期に確立 |
| 外注先の開拓 | M6 から夫婦が外注先を見つけて関係を構築 |
| ソシャゲ運用経験 | 現職の経験が品質判断とディレクションに活きる |

## 夫婦が専業に転向するタイミング（夫婦の判断）

kaedevn 側から「辞めて」とは言わない。夫婦が自分で判断する。

| 条件 | 時期の目安 |
|------|-----------|
| 月間発注額が安定して100万超 | M15頃 |
| 半年以上発注が途切れていない | M12頃 |
| 本業より kaedevn のほうが楽しい/稼げる | 夫婦の主観 |

**M12-M18 のどこかで自然に専業になる可能性が高い。** 発注額が月160-350万あれば、夫婦2人の収入として十分。

## 結論

| 問い | 答え |
|------|------|
| 雇用する？ | **しない。全期間業務委託** |
| なぜ？ | **社保 ¥0、柔軟、夫婦が本業を辞めなくていい、利益率維持** |
| いくらかかる？ | **アセット予算全額（M6: 30万 → M36: 714万）** |
| 何をやってもらう？ | **アセットの全権委任（制作・外注管理・品質管理・予算配分）** |
| 自分の関与は？ | **月1回の方針指示と予算承認のみ** |
| いつ専業になる？ | **夫婦が自分で判断。M12-M18 頃の可能性** |
| ソシャゲ経験は活きる？ | **アセットの品質判断 + 運営型ゲームアプリの助言** |
