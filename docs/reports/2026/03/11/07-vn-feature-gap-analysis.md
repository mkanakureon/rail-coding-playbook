## 5. ブラウザ版の強みと注意点

### 強み（Switch にない利点）
- **URL でアクセス即プレイ** — インストール不要、共有が容易
- **Web Share API** — スクショ + テキストを SNS にワンタップ共有
- **PWA** — ホーム画面追加でアプリ同等体験
- **動画再生** — `<video>` タグで簡単実装（コーデック問題なし）
- **アップデート即反映** — 審査不要、ホットフィックス容易

### ブラウザ固有の注意点
- **音声自動再生制限** — ユーザー操作前に BGM 再生不可（タイトルでクリック誘導が必要）
- **IndexedDB 消失リスク** — ブラウザのデータクリアでセーブ全消失（将来はクラウドセーブで対策）
- **メモリ制限** — モバイル Safari は特にメモリ上限が厳しい（アセットの遅延読み込み必須）
- **フルスクリーン制約** — Fullscreen API はユーザー操作起点のみ
- **戻るボタン対策** — Android の戻るボタンでゲームが閉じないようにする
