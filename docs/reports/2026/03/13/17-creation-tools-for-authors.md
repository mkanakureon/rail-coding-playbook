# ゲーム制作ツール一覧

kaedevn でノベルゲームを作るために使えるツール・機能の全一覧です。「何ができるか」を一目で把握できるカタログとして使ってください。

---

## ブロックエディタ（メイン制作ツール）

ブラウザで動く Web エディタです。ブロックを並べるだけでシナリオが作れます。

| 機能 | 説明 |
|------|------|
| **14種類のブロック** | テキスト、背景、キャラ、選択肢、変数、条件分岐、ジャンプ、エフェクト、フィルタ、タイムライン、オーバーレイ、バトル、スクリプト |
| **ドラッグ&ドロップ** | ブロックの並べ替え、アセットの追加がマウス操作で完結 |
| **リアルタイムプレビュー** | 再生ボタンを押すだけで作品をその場で確認 |
| **レスポンシブ対応** | PC（3カラム）/ タブレット（2カラム）/ スマホ（1カラム）で自動最適化 |
| **アンドゥ・リドゥ** | 最大50回分の操作履歴 |
| **自動保存** | 60秒ごとにサーバーに自動保存 |
| **ページ管理** | 複数ページに分けてシナリオを整理。ページ間ジャンプで物語を分岐 |

**対応ブラウザ**: Chrome / Safari / Firefox / Edge（スマホ含む）
**URL**: ログイン後、マイページ → プロジェクト → エディタ

→ 詳細: [エディタ完全ガイド](/docs/editor-guide) / [ブロック型リファレンス](/docs/block-types) / [ブロック操作ガイド（図解）](/docs/block-guide)

## スクリプトエディタ（上級者向け）

エディタの「Script」タブで、KS / KSC スクリプトを直接記述できます。

| 機能 | 説明 |
|------|------|
| **Kaede Script** | `@bg room` `@ch hero smile R` のようなコマンドで演出を記述 |
| **双方向同期** | スクリプトの変更がブロックに即反映。逆も同様 |
| **一括入力** | 大量のセリフを一気にコピペで投入 |
| **変数・条件式** | `@set score += 10` `@if score > 50` などJS風の構文 |
| **KSC（上級）** | サブルーチン、ループ、関数呼び出しが可能な拡張構文 |

→ 詳細: [ハイブリッドエディタの仕組み](/docs/hybrid-editor) / [KS スクリプト仕様](/docs/ks-script-spec) / [KSC スクリプト仕様](/docs/ksc-script-spec)

## アセット管理

作品に使う素材（背景・立ち絵・BGM・SE・ボイス）をエディタ内で管理します。

| 機能 | 説明 |
|------|------|
| **公式アセット 600点** | ファンタジー・学園・和風などの背景とキャラクター。無料で使用可能 |
| **自作素材アップロード** | PNG / JPG / OGG / MP3 をドラッグ&ドロップで追加 |
| **キャラクター管理** | キャラごとに名前・立ち絵・表情差分をまとめて管理 |
| **カテゴリ分類** | 背景(bg)、キャラ(ch)、オーバーレイ(ovl)、BGM、SE、ボイスに自動分類 |
| **セマンティック検索** | 「夕暮れの教室」のような自然言語でアセットを検索 |

→ 詳細: [アセット管理ガイド](/docs/asset-management) / [キャラクター設定ガイド](/docs/character-setup)

## タイムラインエディタ

キーフレームアニメーションで高度な演出を作れます。

| 機能 | 説明 |
|------|------|
| **トラック編集** | 背景・キャラ・オーバーレイの位置・拡大・透明度を時間軸で制御 |
| **キーフレーム** | 任意のタイミングに値を打ち、自動で補間 |
| **イージング** | linear / easeIn / easeOut / easeInOut から選択 |
| **複数トラック同時** | 背景パンしながらキャラが移動、のような複合演出 |
| **プレビュー連動** | タイムライン再生がそのままプレビューに反映 |

→ 詳細: [タイムラインブロック詳細ガイド](/docs/timeline-guide)

## エフェクト・フィルタ

ブロック1つで画面演出を追加できます。

### エフェクト（effect ブロック）

| 種類 | 効果 |
|------|------|
| shake | 画面を揺らす（地震、衝撃） |
| flash | 画面を一瞬白く光らせる |
| fade | フェードイン / フェードアウト |
| vignette | 画面の周囲を暗くする |
| blur | ぼかし |

### フィルタ（screen_filter ブロック）

| 種類 | 効果 |
|------|------|
| sepia | セピア調（回想シーン向け） |
| grayscale | モノクロ |
| blur | 背景ぼかし |
| crt | ブラウン管テレビ風 |
| gameboy | ゲームボーイ風4色 |
| pc98 | PC-98 風レトロ |

RPG風のバトルをブロック1つで組み込めます。

| 機能 | 説明 |
|------|------|
| **敵グループ設定** | バトルに出現する敵を指定 |
| **勝敗分岐** | 勝利時・敗北時に別ページへジャンプ |
| **ノベル統合** | バトル前後にシナリオを自然に接続 |

→ 詳細: [バトルシステム](/docs/battle-system)

## マップエディタ

RPG風のフィールドマップを作成・編集できます。

| 機能 | 説明 |
|------|------|
| **タイルベース配置** | グリッド上にタイルを並べてマップを構築 |
| **複数レイヤー** | 地面・障害物・装飾を別レイヤーで管理 |
| **マップ間接続** | ワープポイントで複数マップをつなげる |
| **プレビュー** | 作成したマップをブラウザで即確認 |

## AI 執筆支援

AI があなたのアイデアからシナリオの骨格を自動生成します。

| 機能 | 説明 |
|------|------|
| **4段階パイプライン** | 章プロット → 話プロット → シーンテキスト → エディタ形式に段階的に生成 |
| **ジャンルルール** | ファンタジー・恋愛・ミステリー・ホラー等、ジャンルに最適化された生成 |
| **キャラ一貫性** | 登録したキャラクターの口調・性格を反映 |
| **エディタ直接反映** | 生成結果がそのままエディタに入る。コピペ不要 |
| **対話的制作** | 「もっと切なくして」「選択肢を追加して」と会話で修正 |

### AI プロバイダ

| プロバイダ | 用途 |
|-----------|------|
| Claude（Anthropic） | 文章品質の高いシーンテキスト生成・対話的制作 |
| Gemini（Google） | プロット設計・大量シーンのバッチ生成 |
| GPT（OpenAI） | 章構成のブレインストーミング |

→ 詳細: [AI 執筆支援](/docs/ai-writing-assist)

## プレイヤーエンジン（PixiJS）

作品を再生するゲームエンジンです。作者がインストールするものはありません。

| 機能 | 説明 |
|------|------|
| **ブラウザ再生** | PC・スマホ・タブレット、どのブラウザでもそのまま動く |
| **タイトル画面** | はじめから / つづきから / セーブ&ロード / 設定 |
| **セーブ&ロード** | 20スロット + オートセーブ（60秒ごと） |
| **クラウドセーブ** | ログイン中はサーバーに保存。別端末で続きが遊べる |
| **AUTO / SKIP** | 自動送り・既読スキップ |
| **バックログ** | 過去のテキストを遡って読み返し |
| **設定** | テキスト速度、AUTO速度、フォントサイズ、ウィンドウ透明度、色覚モード |
| **PWA** | ホーム画面に追加でアプリ風に動作 |
| **モバイル操作** | タップ、スワイプ、長押しに対応 |
| **NVL モード** | 全画面テキスト表示（小説風） |

→ 詳細: [プレイヤー操作ガイド](/docs/player-guide) / [エンジンの仕組み](/docs/how-engine-works)

| 機能 | 説明 |

| **ワンクリック公開** | プロジェクト設定の「公開スイッチ」をONにするだけ |

## UIレイアウト編集（開発中）

プレイ画面の UI（メッセージウィンドウ、ボタン配置など）をビジュアルに編集できるツールです。

| 機能 | 説明 |
|------|------|
| **プリセット切替** | ノベル標準 / RPG寄り / メッセージ上 / メッセージ中央 |
| **位置・サイズ変更** | メッセージウィンドウ、選択肢、クイックメニューの配置を自由に調整 |
| **透明度・フォント** | ウィンドウの透明度やフォントサイズをプロジェクトごとに設定 |
| **JSON出力** | レイアウト定義が JSON で保存される（git管理・AI生成が可能） |
| **Web / Switch 共通** | 同じ設定が Web 版と Switch 版の両方に適用される |

→ 現在開発中。プレビュー版: [レイアウトプレビュー](/docs/tsukuru-layout-preview)

## 制作環境

| 項目 | 内容 |
|------|------|
| **必要なもの** | ブラウザだけ。インストール不要 |
| **対応OS** | Windows / Mac / Linux / iPad / スマホ |
| **アカウント** | 無料登録（メールアドレスのみ）。ゲストモード（72時間）もあり |

| **データ形式** | JSON（git管理可能、人間が読める、AI生成可能） |
| **保存先** | クラウド（サーバー自動保存） |

## はじめ方

1. [kaedevn にアクセス](/) してアカウント登録（またはゲストで開始）
2. マイページで「新規プロジェクト」をタップ
3. 自動で背景・キャラクター・テキストが入った状態でエディタが開く
4. ブロックを追加・編集してシナリオを作る
5. 再生ボタンでプレビュー
6. 公開スイッチを ON にして URL を共有

→ 詳細: [制作の第一歩](/docs/getting-started) / [スマホで作る・遊ぶ](/docs/mobile-guide)
