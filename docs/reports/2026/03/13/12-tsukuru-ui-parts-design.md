# ツクール型エディタ — UI 部品デザインガイド

- **作成日**: 2026-03-13
- **親文書**: `tsukuru-editor-spec.md`
- **用途**: 各UI部品の見た目・振る舞い・制約・既存実装との対応

---

## 1. 概要

このドキュメントは全20種のUI部品について、以下を定義する：

- **見た目**: サイズ・色・フォント・装飾
- **振る舞い**: アニメーション・表示条件・インタラクション
- **制約**: 配置制限・サイズ制限・依存関係
- **既存実装との対応**: 現行コードのどこに相当するか

## 2. ノベル基本 UI（Phase 1）

### 2-1. message-window（メッセージウィンドウ）

**既存実装**: `packages/web/src/renderer/TextWindow.ts`

| 項目 | 値 |
|------|-----|
| デフォルトサイズ | 1200 × 180 px |
| 最小サイズ | 800 × 120 px |
| 最大サイズ | 1280 × 600 px（NVL時） |
| 背景 | 半透明黒 `rgba(0,0,0,0.7)` |
| テキスト色 | `#ffffff` |
| フォントサイズ | 24px（本文）/ 20px（名前） |
| パディング | 20px |
| 角丸 | 8px |

**振る舞い**:
- テキスト送り（タイプライター表示）、速度はユーザー設定
- AUTO / SKIP モード時にステータスラベル表示
- ADV モード: 固定高さ（180px）、新テキストで入れ替え
- NVL モード: 可変高さ（最大600px）、テキスト追記
- 非表示ボタン押下で `visible: false`（背景鑑賞用）

**制約**:
- 画面内に収まること（左上原点 + サイズ ≤ 1280×720）
- サイズ変更は Phase 3 以降
- フォントサイズ変更は Phase 3 以降（設定画面から変更可能にする）

**内部構造**:
```
message-window (Container)
├ background (Graphics)       ← 半透明矩形
├ nameText (Text)             ← 話者名（name-box が外部の場合は非表示）
├ bodyText (Text)             ← 本文テキスト
├ statusLabel (Text)          ← AUTO/SKIP 表示
└ clickWaitIcon (Container)   ← 内部配置の場合
```

### 2-2. name-box（名前欄）

**既存実装**: `TextWindow.ts` 内の `nameText`（現在はウィンドウ内部に統合）

| 項目 | 値 |
|------|-----|
| デフォルトサイズ | 200 × 30 px |
| 最小サイズ | 80 × 24 px |
| 最大サイズ | 400 × 40 px |
| 背景 | 半透明黒 `rgba(0,0,0,0.8)` |
| テキスト色 | `#ffffff` |
| フォントサイズ | 20px |
| 角丸 | 4px |

**振る舞い**:
- 話者名がある時のみ表示（`who` パラメータ）
- 名前が空の時は自動で `visible: false`
- テキスト内容に応じて幅を自動調整（最小80px〜最大400px）

**配置パターン**（Phase 3 で選択可能に）:
| パターン | 位置 | 説明 |
|---------|------|------|
| above-left | メッセージウィンドウの左上 | 標準（ノベル/RPG） |
| below-left | メッセージウィンドウの左下 | メッセージ上配置時 |
| inside-top | メッセージウィンドウ内上部 | NVL モード向け |

**制約**:
- message-window に追従（message-window 移動時に連動）
- 単体での位置変更は Phase 3 以降

### 2-3. choice-window（選択肢ウィンドウ）

**既存実装**: `packages/web/src/renderer/ChoiceOverlay.ts`

| 項目 | 値 |
|------|-----|
| デフォルトサイズ | 500 × 300 px |
| ボタンサイズ | 400 × 50 px |
| ボタン間隔 | 12px |
| 背景暗転 | `rgba(0,0,0,0.4)` |
| ボタン色 | `rgba(40,40,80,0.9)` |
| ホバー色 | `rgba(60,60,120,0.9)` |
| テキスト色 | `#ffffff` |
| フォントサイズ | 22px |

**振る舞い**:
- `choice` コマンド時のみ表示
- 選択完了で自動非表示
- キーボード操作: ↑↓ で移動、Enter で決定
- マウス: ホバーでフォーカス、クリックで決定
- フォーカス中のボタンは色変更 + 微拡大（1.05倍）
- 表示時にフェードイン（300ms）

**制約**:
- 常に画面中央付近（大きくずらさない）
- 選択肢数に応じて高さ自動調整
- 最大選択肢数: 8

### 2-4. click-wait-icon（クリック待ちマーク）

| 項目 | 値 |
|------|-----|
| サイズ | 24 × 24 px |
| 色 | `#ffffff` |
| 形状 | 逆三角形 |
| アニメーション | パルス（上下にバウンス、0.05速度） |

**振る舞い**:
- テキスト表示完了時に出現
- ユーザーがクリック/タップすると消える
- AUTO モード時は非表示（自動送りのため）

**制約**:
- **常にメッセージウィンドウ内の右下**に配置
- 単体での位置変更不可（message-window に追従）
- エディタでは位置変更できない（Phase 1〜4 共通）

### 2-5〜2-9. クイックメニューボタン群

| ボタン | ラベル | アクティブ色 | 機能 |
|--------|--------|-------------|------|
| auto-button | AUTO | `#4fc3f7`（水色） | 自動テキスト送り ON/OFF |
| skip-button | SKIP | `#ff8a65`（オレンジ） | 既読スキップ ON/OFF |
| log-button | LOG | `#aaaaaa` | バックログ画面を開く |
| hide-ui-button | HIDE | `#aaaaaa` | UI 一時非表示 |
| menu-button | MENU | `#aaaaaa` | システムメニューを開く |

**共通仕様**:
| 項目 | 値 |
|------|-----|
| サイズ | 48 × 32 px |
| フォントサイズ | 12px |
| 非アクティブ色 | `#888888` |

**振る舞い**:
- AUTO / SKIP はトグル式（タップで ON↔OFF）
- ON 状態ではアクティブ色で表示
- LOG / HIDE / MENU はタップで画面遷移 or 状態変更
- ホバーで微透明度アップ

**制約**:
- quick-menu-bar 内に等間隔で配置
- ボタンの表示/非表示は Phase 3 で選択可能に

## 3. 通知・補助 UI（Phase 2）

### 3-1. toast-notification（通知トースト）

| 項目 | 値 |
|------|-----|
| デフォルトサイズ | 400 × 48 px |
| 背景 | `rgba(0,0,0,0.8)` |
| テキスト色 | `#ffffff` |
| フォントサイズ | 16px |
| 角丸 | 24px（ピル型） |
| 表示時間 | 3000ms |
| フェードイン | 300ms |
| フェードアウト | 500ms |

**振る舞い**:
- プログラムから `showToast(text)` で呼び出し
- 複数同時には出さない（キュー方式、前のが消えてから次を表示）
- フェードイン → 表示時間 → フェードアウト
- タイマー動作中の新通知は前のを即消して新しいのを表示

**用途例**:
- 「セーブしました」
- 「オートモード ON」
- 「実績解除: はじめての選択」

## 5. zIndex 設計

UI 部品間の重なり順を明確に定義する。

| 範囲 | カテゴリ | 部品 |
|------|---------|------|
| 100–119 | テキスト系 | message-window(100), name-box(110), click-wait-icon(115) |
| 120–139 | 情報表示 | objective-panel(125), quest-tracker(124), area-name-plate(130), mini-map(130), party-status-panel(135) |
| 140–149 | 通貨 | gold-window(140) |
| 150–159 | メニュー | quick-menu-bar(150), ボタン群(151), interact-hint(160) |
| 200–299 | モーダル | choice-window(200) |
| 300–319 | 通知 | toast-notification(300), save-indicator(310) |
| 500+ | 全画面 | loading-overlay(500) |

**ルール**:
- モーダル（choice-window）は他の全UIの上
- loading-overlay は最上位
- 通知系は情報表示の上
- メニューボタンはテキストの上

## 6. 共通デザイントークン

全 UI 部品で共通のデザイン値。ウィンドウスキン対応（Phase 2）で差し替え可能にする。

```typescript
const UI_TOKENS = {
  // 背景
  bgPrimary: "rgba(0,0,0,0.7)",
  bgSecondary: "rgba(0,0,0,0.5)",
  bgOverlay: "rgba(0,0,0,0.4)",

  // テキスト
  textPrimary: "#ffffff",
  textSecondary: "#cccccc",
  textAccent: "#ffd700",
  textActive: "#4fc3f7",
  textWarning: "#ff8a65",

  // バー
  hpColor: "#4caf50",
  mpColor: "#42a5f5",
  xpColor: "#ffd700",

  // 角丸
  radiusSmall: 4,
  radiusMedium: 8,
  radiusLarge: 18,    // ピル型

  // フォント
  fontBody: 24,
  fontSub: 20,
  fontSmall: 16,
  fontMicro: 12,
  fontInfo: 13,

  // アニメーション
  fadeInMs: 300,
  fadeOutMs: 500,
  slideInMs: 400,
} as const;
```
