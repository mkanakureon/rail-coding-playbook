# フィルタースライダー機能 説明書

> Generated by Claude Opus 4.6

## 概要

エディタのスクリーンフィルターブロックに、フィルター別のパラメータ調整スライダーを追加しました。
従来は「フィルター種別」と「強度（0〜1）」しか設定できませんでしたが、各フィルターが持つ個別のパラメータ（速度・角度・ぼかし半径など）をスライダーで直感的に調整できるようになりました。

### Before / After

**Before（従来）:**
```
[▼ 雨]  強度 [ 0.5 ]
```

**After（新機能）:**
```
[▼ 雨]    強度 [===●===] 0.5
速度       [======●==] 2.0
角度       [===●=====] 0.1
雨粒の長さ  [==●======] 0.15
                    [リセット]
```

---

## 操作方法

### 1. フィルターを選択する

ブロックカード左側のドロップダウンからフィルターを選択します。

### 2. 強度を調整する

全フィルター共通の「強度」スライダー（0.0〜1.0）で、フィルター全体の効果の強さを調整します。

### 3. 個別パラメータを調整する

フィルターを選択すると、そのフィルター固有のスライダーが自動的に表示されます。
各スライダーはレンジバー（ドラッグ）と数値入力（直接入力）の両方で操作できます。

### 4. リセット

パラメータをデフォルト値から変更した場合、「リセット」ボタンが表示されます。
クリックすると全パラメータがデフォルト値に戻ります。

### 5. フィルター切り替え

フィルター種別を変更すると、前のフィルターのパラメータは自動的にクリアされます。

## 対応フィルター一覧

### 特殊演出系

#### ビネット（vignette）
画面の周辺を暗くする演出。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| intensity | 強度 | 0〜1 | 0.5 | 暗くなる度合い |
| radius | 範囲 | 0〜1 | 0.75 | 暗くなり始める位置（0=中心から、1=端だけ） |
| softness | 柔らかさ | 0〜1 | 0.45 | 暗くなる境界のぼかし具合 |

#### 夜（night）
画面全体を暗く青みがかった色合いにする。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| intensity | 強度 | 0〜1 | 0.6 | 全体の効果の強さ |
| darkness | 暗さ | 0〜1 | 0.5 | 暗くする度合い |
| blueShift | 青みシフト | 0〜1 | 0.3 | 青色にシフトする度合い |

#### ブルーム（bloom）
明るい部分を光らせてにじませる演出。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| intensity | 強度 | 0〜1 | 0.5 | 光のにじみの強さ |
| threshold | しきい値 | 0〜1 | 0.6 | 光り始める明るさの基準（低い=多くの部分が光る） |
| radius | ぼかし半径 | 1〜8 | 4.0 | 光のにじみの広がり |

#### ノイズ（noise）
画面にザラザラしたノイズを追加する。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| intensity | 強度 | 0〜1 | 0.5 | ノイズの量 |
| scanlines | 走査線 | 0〜1 | 0.3 | 横縞の走査線エフェクトの強さ |

#### グリッチ（glitch）
デジタルノイズのような画面の乱れ。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| intensity | 強度 | 0〜1 | 0.5 | 乱れの激しさ |
| blockSize | ブロックサイズ | 0.01〜0.2 | 0.05 | 乱れるブロックの大きさ（小さい=細かい乱れ） |

#### フォーカスブラー（focusBlur）
中心はくっきり、周辺をぼかす被写界深度演出。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| intensity | 強度 | 0〜1 | — | ぼかし量（blurAmount と連動） |
| focusRadius | フォーカス半径 | 0.05〜0.8 | 0.3 | くっきり見える範囲の広さ |
| blurAmount | ぼかし量 | 0.5〜10 | 3.0 | 周辺のぼかしの強さ |

#### カラーティント（colorTint）
画面全体に色を重ねる。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| intensity | 強度 | 0〜1 | 0.3 | 色の重ね具合 |
| brightness | 明るさ | 0〜2 | 1.0 | 明るさの調整（1.0=変化なし） |

#### 色収差（chromaticAberration）
RGB の色ずれを再現するレンズ演出。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| intensity | 強度 | 0〜1 | 1.0 | 色ずれの強さ |
| offset | ずれ量 | 0〜0.02 | 0.005 | 色がずれるピクセル距離 |

#### 水中（underwater）
水中にいるような波打つ演出。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| waveIntensity | 波の強さ | 0〜0.05 | 0.01 | 画面がゆらゆら揺れる度合い |
| waveSpeed | 波の速度 | 0.1〜3 | 1.0 | 揺れの速さ |
| tintIntensity | 色合い | 0〜1 | 0.4 | 青みがかった色の強さ |
| causticIntensity | コースティクス | 0〜0.5 | 0.15 | 水面の光の模様の強さ |

### レトロ系

#### CRT（crt）
ブラウン管モニターの表示を再現。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| curvature | 曲面歪み | 0〜0.1 | 0.03 | 画面の丸み（ブラウン管の膨らみ） |
| scanlineStrength | 走査線 | 0〜0.5 | 0.15 | 横縞の走査線の濃さ |
| phosphorStrength | 蛍光体 | 0〜1 | 0.3 | RGB 蛍光体の光の強さ |
| vignette | 周辺減光 | 0〜1 | 0.3 | 画面の四隅が暗くなる度合い |

#### オールドフィルム（oldFilm）
古い映画フィルムの質感を再現。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| grainIntensity | グレイン | 0〜1 | 0.15 | フィルムの粒子感 |
| flickerIntensity | ちらつき | 0〜0.1 | 0.03 | 画面がちらつく度合い |
| sepiaStrength | セピア | 0〜1 | 0.5 | セピア色の強さ |
| scratchDensity | 傷密度 | 0〜1 | 0.3 | フィルムの傷の多さ |

#### PC-98（pc98）
PC-98 の 16色アナログパレットを再現。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| ditherStrength | ディザ強度 | 0〜0.5 | 0.15 | ディザリングパターンの目立ち具合 |
| saturation | 彩度 | 0.5〜3 | 1.5 | 色の鮮やかさ（16色に減色する前に適用） |
| gamma | ガンマ | 0.5〜1.5 | 0.95 | 明暗カーブの補正 |
| scanlineStrength | 走査線 | 0〜0.2 | 0.08 | 走査線エフェクトの強さ |
| (checkbox) | タイリング | on/off | off | Bayer 8x8 ディザで 2色を交互配置して擬似色増加 |

#### ゲームボーイ（gameboy）
ゲームボーイの 4色緑パレットを再現。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| contrast | コントラスト | 0.5〜3 | 1.0 | 4色の区別のしやすさ（低い=のっぺり、高い=くっきり） |

#### ファミコン（famicom）
NES/ファミコンの 52色パレット + ドット絵を再現。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| pixelSize | ドットサイズ | 2〜16 | 4 | 1ドットの大きさ（大きい=粗い） |
| scanlineStrength | 走査線 | 0〜0.2 | 0.06 | 走査線の濃さ |

#### ピクセル化（pixelate）
単純なピクセル化（色数制限なし）。

| パラメータ | ラベル | 範囲 | デフォルト | 説明 |
|-----------|--------|------|-----------|------|
| pixelSize | ドットサイズ | 2〜32 | 8 | 1ドットの大きさ |

| 天候 | 曇天 |

## KSC スクリプトでの記法

### 基本構文

```
@filter <フィルター名> [強度] [パラメータ名=値] ...
```

### 例

```ksc
; 雨フィルター：強度0.4、速度2倍、やや左傾き
@filter rain 0.4 speed=2.0 angle=-0.1

; CRT：走査線を強めに
@filter crt scanlineStrength=0.3 phosphorStrength=0.5

; PC-98：タイリング有効、彩度高め
@filter pc98 0.5 tiling saturation=2.0

; ファミコン：粗いドット
@filter famicom pixelSize=8

; フィルター解除
@filter_clear
```

### 後方互換性

従来の書き方はそのまま動作します。

```ksc
; 従来の書き方（引き続き有効）
@filter sepia 0.8
@filter rain
@filter pc98 0.5 tiling
```

## 品質レベルとの関係

品質レベル（high / medium / low）によって一部フィルターの動作が変わります。
スライダーで設定したパラメータは **high / medium** で使用されますが、**low** ではフォールバック（代替フィルターまたは無効化）が優先されます。

| フィルター | high | medium | low |
|-----------|------|--------|-----|
| bloom | パラメータ有効 | radius 縮小 | vignette にフォールバック |
| rain | パラメータ有効 | intensity 半減 | night にフォールバック |
| snow/sakura | パラメータ有効 | intensity 半減 | 無効 |
| crt | パラメータ有効 | curvature=0 固定 | 無効 |
| glitch | パラメータ有効 | blockSize 拡大 | 無効 |
| oldFilm | パラメータ有効 | flicker/scratch 無効 | sepia のみ |
| underwater | パラメータ有効 | caustic 無効 | colorTint のみ |
| chromaticAberration | パラメータ有効 | offset 半減 | 無効 |

## 技術仕様

### データ構造

```typescript
type ScreenFilterBlock = {
  id: string;
  type: 'screen_filter';
  filterType: string;               // フィルター種別
  intensity?: number;               // 強度 0.0〜1.0
  pc98Tiling?: boolean;             // PC-98 タイリングモード
  params?: Record<string, number>;  // フィルター別パラメータ
};
```

### パラメータ定義の場所

`apps/editor/src/components/blocks/filterParamDefs.ts`

新しいフィルターを追加する場合、この定義ファイルにエントリを追加するだけでスライダーが自動的に表示されます。

### パイプライン

```
Editor UI (sliders)
  → ScreenFilterBlock.params に保存
  → getBlockScript() で KSC 出力: @filter rain 0.4 speed=2.0
  → compiler parseFilterCommand() でパース: Op { options: { speed: 2.0 } }
  → ScreenFilter.createFilter(type, intensity, options) でフィルター生成
  → WebGL シェーダーに反映
```
