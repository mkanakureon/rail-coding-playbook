## 実装項目

### 1. `--json` フラグ（全 CLI コマンド）

**目的**: Claude Code が CLI 出力を構造的に処理できるようにする

**方針**:
- `context.ts` にグローバル `--json` フラグ検出を追加
- 各コマンドの `handler` に `json` モードの出力分岐を追加
- `types.ts` の `Command` 型に `jsonOutput` を追加は不要 — 各 handler 内で `isJson()` をチェックするだけ

**対象コマンド（32個）**:

| 優先 | コマンド | JSON 出力内容 |
|:----:|---------|-------------|
| 高 | `create` | `{ id, type, name, status: "created" }` |
| 高 | `list` | `[{ id, type, name, properties }]` |
| 高 | `get` | `{ id, type, name, properties, spatial, relations }` |
| 高 | `set` | `{ id, key, value, status: "updated" }` |
| 高 | `delete` | `{ id, status: "deleted" }` |
| 高 | `create-map` | `{ id, name, width, height }` |
| 高 | `place-event` | `{ id, name, mapId, x, y, trigger }` |
| 高 | `add-command` | `{ eventId, commandType, status: "added" }` |
| 中 | `search` | `[{ id, type, name }]` |
| 中 | `clone` | `{ originalId, cloneId, name }` |
| 中 | `stats` | `{ entities, relations, schemas, byType }` |
| 中 | `analyze` | `[{ type, entityId, message }]` |
| 中 | `balance` | `{ results: [...], warnings: [...] }` |
| 中 | `compile` | `{ events: [{ id, name, instructions }] }` |
| 中 | `validate` | `{ schemaErrors: [...], analysisIssues: [...] }` |
| 低 | その他 | 同様のパターン |

**実装パターン**:

```typescript
// context.ts に追加
export function isJson(): boolean {
  return process.argv.includes("--json");
}

export function output(data: unknown): void {
  if (isJson()) {
    console.log(JSON.stringify(data));
  }
}

// 各コマンドでの使い方
handler: async (args) => {
  await withAPI((api) => {
    const entity = api.createEntity(args[0], args[1]);
    if (isJson()) {
      output({ id: entity.id, type: entity.type, name: entity.name, status: "created" });
    } else {
      console.log(`Created: ${entity.id} (${entity.type}) "${entity.name}"`);
    }
  });
}
```

### 2. プレビュー確認スキル

**目的**: Claude Code がゲーム画面のスクリーンショットを撮って視覚確認できる

**方針**: 新しい Claude Code スキル `rpg-preview` を作成。Playwright MCP を使って：

1. `rpg-cli sync-export` で JSON を出力
2. エディタ（localhost:5178）で Import
3. Play Game タブでスクリーンショット撮影
4. Claude Code に画像を返す

**スキルトリガー**: `"RPGプレビュー"`, `"ゲーム画面確認"`, `"rpg preview"`

**スキルファイル**: `.claude/skills/rpg-preview/`
