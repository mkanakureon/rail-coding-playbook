# エディタ アップデート内容 (2026-03-06)

## 間違えても戻せるようになった

- `Ctrl+Z`（Mac: `Cmd+Z`）で直前の操作を取り消し
- `Ctrl+Shift+Z` または `Ctrl+Y` でやり直し
- ヘッダーの ↩ ↪ ボタンでも同じ操作ができる
- ブロック追加・削除・移動・並べ替え・編集、ページ追加・削除、すべて50回分まで戻れる
- テキスト入力やスライダー操作は、手を止めてから0.5秒後にまとめて1回分として記録される（連打で履歴が溢れない）

## ブロックをコピーできるようになった

- 各ブロックカードの右側に ⧉（複製）ボタンが表示される（PC のみ）
- `Ctrl+D`（Mac: `Cmd+D`）でも複製できる
- 複製されたブロックは元ブロックの直後に挿入される
- IF文や選択肢など、中にブロックを含むものもまるごとコピーされる
- 複製も `Ctrl+Z` で取り消せる
- START ブロックは複製不可（ページに1つだけのルール）

## 背景・キャラ・OVLの位置調整が「実行」にも反映されるようになった

- X / Y / スケールのスライダーで調整した値が、サイドバーのプレビューだけでなく「実行」ボタンのプレイ画面にも正しく反映される
- これまではプレビューパネルでは位置が変わって見えていたが、実行ボタンを押すとデフォルト位置に戻っていた

## ブロック数が多いページの動作が軽くなった

- 全14種類のブロックカードに React.memo を適用
- あるブロックを編集しているとき、他のブロックが不要に再描画されなくなった
- ブロック数が多いページでスクロールや入力がスムーズになる

## キーボードショートカット一覧

| 操作 | Windows / Linux | Mac |
|------|----------------|-----|
| 元に戻す | `Ctrl+Z` | `Cmd+Z` |
| やり直し | `Ctrl+Shift+Z` / `Ctrl+Y` | `Cmd+Shift+Z` |
| ブロック複製 | `Ctrl+D` | `Cmd+D` |
