## 背景

`./scripts/test-local.sh --phase 6` でローカル E2E テスト（Phase 6: Playwright）を実行したところ、34 テストが失敗した。原因は大きく2つ：

## 教訓

1. **auto-init の導入時はテストへの影響を事前にリストアップする**: プロジェクト作成のデフォルト動作を変えると、既存テストの `beforeAll` で作成したデータと衝突する
2. **UI ラベル変更は横断的にテストを検索する**: `grep` で古いラベルを使っているテストを洗い出してから変更する
3. **Azure テストとローカルテストは config で明確に分離する**: `testIgnore` パターンで Azure 専用テストを除外
4. **fixtures は実環境と一致させる**: `db.ts` のポート番号のような基本設定は定期的に検証する
5. **Monaco エディタのテストは `waitForFunction` でロード完了を待つ**: `@monaco-editor/react` は非同期ロードのため、DOM 出現だけでなくモデル生成まで待つ必要がある
