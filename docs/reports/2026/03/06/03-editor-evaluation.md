## 強み

- **ブロック → KSC → コンパイラ → エンジン** の一気通貫パイプライン
- **リアルタイムプレビュー**: ブロック選択時に即座に postMessage → iframe 描画
- **位置・スケール調整**: X/Y/S スライダー + 即プレビュー（競合ツールにない機能）
- **TypeScript strict mode** 全有効、型安全性が高い
- **レスポンシブ**: モバイル1カラム ↔ デスクトップ3カラム、リサイズ可能サイドバー
- **ダークモード**: `dark:` 509箇所の包括的対応
- **テスト 177件**: ストア CRUD、スクリプト生成、型テスト、API設定テスト
