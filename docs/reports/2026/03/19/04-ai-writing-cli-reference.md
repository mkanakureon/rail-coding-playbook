# AI 執筆支援 CLI リファレンス

> Generated by Claude Opus 4.6

AI によるシナリオ自動生成パイプラインの CLI ツール。
限定公開機能。

## 環境変数

```bash
export GOOGLE_API_KEY=<your-key>    # Gemini API（必須）
export KAEDEVN_API=http://localhost:8080/api
```

---

## 1. シナリオ自動生成（`assist-cli.ts`）

設定ファイルから4段階パイプラインでシナリオを自動生成する。

```bash
# 全段階実行
npx tsx scripts/cli/ai/assist-cli.ts all --settings <project-dir>

# 途中から再開
npx tsx scripts/cli/ai/assist-cli.ts all --settings <dir> --resume <output-dir>

# 特定の章・話だけ
npx tsx scripts/cli/ai/assist-cli.ts all --settings <dir> --chapter 2 --episode 3

# ドライラン（生成せず計画のみ）
npx tsx scripts/cli/ai/assist-cli.ts all --settings <dir> --dry-run

# モック（API 呼び出しなし）
npx tsx scripts/cli/ai/assist-cli.ts all --settings <dir> --mock

# API 使用量ログ
npx tsx scripts/cli/ai/assist-cli.ts all --settings <dir> --log-api
```

### 4段階パイプライン

| Stage | モデル | 内容 |
|-------|--------|------|
| 0 | gemini-2.5-pro | アイデア → 構造化プロット |
| 1 | gemini-2.5-pro | プロット → 章アウトライン |
| 2 | gemini-2.5-flash | 章 → 話サマリー |
| 3-4 | gemini-2.5-flash | 話 → フルテキスト → エディタJSON |

---

## 2. 設定ファイル生成（`upstream-cli.ts`）

アイデアメモから設定ファイル一式を自動生成する。

```bash
# アイデアメモ → 設定ファイル
npx tsx scripts/cli/ai/upstream-cli.ts generate-settings idea.md ./output/
```

生成される設定ファイルは `assist-cli.ts` の `--settings` に渡す。

---

## 3. 生成結果のアップロード

生成されたエディタ JSON をプロジェクトとしてアップロードする。

```bash
node scripts/cli/ks-upload.mjs ./my-story/editor-json/
```

---

## 典型的なワークフロー

```bash
# 1. アイデアメモから設定ファイル生成
npx tsx scripts/cli/ai/upstream-cli.ts generate-settings idea.md ./my-story/

# 2. 設定ファイルを確認・修正（手動）

# 3. シナリオ生成
npx tsx scripts/cli/ai/assist-cli.ts all --settings ./my-story/

# 4. プロジェクトとしてアップロード
node scripts/cli/ks-upload.mjs ./my-story/editor-json/
```
