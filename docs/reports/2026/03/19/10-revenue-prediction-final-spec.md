## エグゼクティブサマリー

kaedevn を「ノベルゲーム制作ツール」ではなく、**「15分完結の物語を売り買いするプラットフォーム」** として位置づける。

収益の柱を「作者向けサブスク」から **「読者向けポイント課金（流通手数料）」** に転換することで、MAU 成長に比例して収益が指数関数的に伸びるモデルになる。

## 1. ビジネスモデルの転換

### 前版（09）: ツール課金モデル

```
作者が月額課金（容量・書き出し）→ kaedevn に直接収益
読者は無料 → 広告収益のみ
```

**問題**: 作者数が頭打ち（8,000〜12,000人）になると収益も頭打ち。

### 本版: プラットフォーム課金モデル

```
読者がポイント購入 → 作品に消費 → kaedevn が手数料10%を徴収
作者は売上の90%を受け取る（業界最高水準）
無課金読者 → 広告 or 「待てば無料」
```

**利点**: 読者数（MAU）に比例して収益が伸びる。天井が高い。マンガアプリと同じモデル。

## 3. コンテンツ仕様：マイクロ VN

### 15分完結の読み切り体験

| 項目 | 仕様 |
|------|------|
| プレイ時間 | 15〜20分（読み切り完結） |
| 価格 | **100 pt（100円相当）** |
| 無料トライアル | 冒頭 5 分間（起承転結の「承」まで） |
| ペイウォール位置 | 最も盛り上がる箇所（クリフハンガー） |
| 画面 | スマホ縦画面（幅720・ボトムアンカーUI） |

### ポイント（コイン）制

| プラン | 価格 | 付与ポイント | 備考 |
|:---|:---|:---|:---|
| お試し | 300円 | 300 pt | 読み切り3本分 |
| 標準 | 500円 | 520 pt | 5本＋20ptボーナス |
| **推奨** | **1,000円** | **1,100 pt** | **10本＋1本無料** |

### ポイント制の利点

1. **決済手数料の最適化**: 100円×10回 → 手数料300円。1,000円×1回 → 手数料30円。利益率が10倍
2. **「ついで買い」の誘発**: 残高がある → 「もう1本読もう」
3. **シームレスな体験**: SNS から流入 → ポイント残高あり → 1タップでアンロック

### 読者の体験フロー

```
TikTok で 15秒の名場面動画を見る
  ↓
リンクタップ → ブラウザで3秒以内に1ページ目表示（ワンタップ起動）
  ↓
5分間の無料プレイで物語に引き込まれる
  ↓
「続きを 100pt で読む」→ 1タップでアンロック
  ↓
読了 → 同じ作者の別作品をレコメンド → もう1本
  ↓
気に入った作者の連載をフォロー → リテンション
```

### 無課金読者の導線

```
無料トライアル（5分）
  ↓
課金したくない → 「24時間待てば次の章が無料」
  ↓
待ち時間に広告表示 → 広告収益
  ↓
待ちきれない → ポイント購入 → 課金転換
```

## 10. 結論

### 作者分配 90% でも年間 3,900 万円が射程圏内

ツクール型エディタを外し、作者に業界最高の 90% を還元しても、**マイクロ VN（100円完結）× ポイント課金 × SNS 縦動画拡散**を組み合わせることで年間 3,900 万円に到達できる。「作者が儲かるプラットフォーム」というブランドが、結果的に MAU を押し上げ、広告と Switch で利益を確保する構造。

### 勝ち筋の公式

```
15分完結 × 100円 × 縦画面 × ブラウザ即プレイ × SNS拡散 × Switch出版
= マンガアプリのUX × ノベルゲームのコンテンツ × Switchのプレミアム感
```

### 最優先アクション

1. **Phase 1（OGP + ワンタップ + シェア）をリリースに含める** — 工数 1 週間
2. **Phase 2 でポイント課金を最速実装** — これが収益のゲームチェンジャー
3. **M3 でゲームジャムを開催** — 100円完結作品を大量に生み出す
4. **M6 で Switch アンソロジー出版** — Web の人気短編をパッケージ化
