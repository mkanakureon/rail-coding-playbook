# Claude Code からノベルゲームを操作する例

> Generated by Claude Opus 4.6

Claude Code に自然言語で指示するだけで、ブラウザのエディタを開かずにノベルゲームのシナリオを制作・編集できる。

## 仕組み

| 方法 | 説明 |
|------|------|
| **スキル経由** | 日本語で指示 → Claude Code がスキル内の手順に従って CLI を実行 |
| **直接指示** | 「このコマンドを実行して」→ Bash ツールで実行 |

どちらも裏では同じ CLI（`editor-cli.ts` 等）を呼んでいる。

### 前提

- ローカル API サーバーが起動していること（`./scripts/dev-start.sh api`）

## 5. アセットを管理する

ローカルの画像をプロジェクトやマイアセット（個人ライブラリ）にアップロードする。

### Claude Code への指示例

```
プロジェクトのアセット一覧を見せて
```

```
この画像をプロジェクトの背景としてアップロードして: ./images/forest.png
```

```

```

```

```

```
この画像をマイアセットに追加して: ./my_bg.png カテゴリは背景
```

### 裏で実行されるコマンド

```bash
# プロジェクトアセット一覧
npx tsx scripts/cli/asset/asset-cli.ts list <projectId>

# ローカル画像をプロジェクトにアップロード
npx tsx scripts/cli/asset/asset-cli.ts upload <projectId> ./images/forest.png --category bg

# ディレクトリ一括アップロード
npx tsx scripts/cli/asset/asset-cli.ts upload-dir <projectId> ./kaede_faces/ --category ch-img

# マイアセット一覧
npx tsx scripts/cli/asset/asset-cli.ts my-list

# マイアセットに追加
npx tsx scripts/cli/asset/asset-cli.ts my-upload ./my_bg.png --category bg

# マイアセットからプロジェクトにインポート
npx tsx scripts/cli/asset/asset-cli.ts import-library <projectId> <userAssetId>
```

## 6. 選択肢・分岐を作る

### Claude Code への指示例

```
「戦う」と「逃げる」の選択肢を追加して。
「戦う」はバトルページ、「逃げる」はエスケープページにジャンプ
```

```
「バッドエンド」ページを追加して、暗い背景に「ここで終わりだ」というセリフを入れて
```

### 裏で実行されるコマンド

```bash
# 選択肢
npx tsx scripts/cli/block/editor-cli.ts add <projectId> <pageId> choice \
  --text "どうする？" \
  --options '["戦う","逃げる"]'

# ページ追加 + ブロック追加
npx tsx scripts/cli/block/editor-cli.ts add-page <projectId> "バッドエンド"
npx tsx scripts/cli/block/editor-cli.ts add <projectId> <newPageId> bg --assetId <暗い背景id>
npx tsx scripts/cli/block/editor-cli.ts add <projectId> <newPageId> text \
  --speaker "" --body "ここで終わりだ"
```

## 7. キャラクターを作成する

公式アセットまたはマイアセットの画像を使って、プロジェクトにキャラクターを新規作成する。
公式アセットの場合は表情差分（笑顔・怒り等）が自動取得される。
マイアセットの場合は asset ID を直接指定して表情を割り当てる。

### Claude Code への指示例

```
公式アセットにどんなキャラがいるか見せて
```

```
楓のキャラクターを作成してプロジェクトに追加して
```

```
マイアセットの画像を使って魔王のキャラクターを作って
```

```
プロジェクトのキャラ一覧を見せて
```

### 裏で実行されるコマンド

```bash
# 公式アセットのキャラクター一覧を確認
npx tsx scripts/cli/character/character-cli.ts official

# 公式アセットの画像でキャラクターを新規作成（表情差分も自動取得）
npx tsx scripts/cli/character/character-cli.ts create-from-official <projectId> \
  --slug kaede --name 楓 --subcategory 楓

# マイアセットの画像でキャラクターを新規作成（asset ID を直接指定）
npx tsx scripts/cli/character/character-cli.ts create <projectId> \
  --slug villain --name "魔王" \
  --expressions '{"angry":"asset-id-1","laugh":"asset-id-2"}'

# プロジェクト内のキャラクター一覧
npx tsx scripts/cli/character/character-cli.ts list <projectId>
```

## 10. 一連の流れの実例

### 「ファンタジー短編を1本作って」と指示した場合

Claude Code が自動で行う手順:

```bash
# 1. プロジェクト作成
npx tsx scripts/cli/project/init-project.ts --config fantasy --title "勇者の旅立ち"

# 2. キャラクター作成
npx tsx scripts/cli/character/character-cli.ts create-from-official <projectId> \
  --slug kaede --name 楓 --subcategory 楓

# 3. 利用可能アセット確認
npx tsx scripts/cli/block/editor-cli.ts context <projectId>

# 4. シーン構築（背景 → キャラ → セリフ × N → 選択肢）
npx tsx scripts/cli/block/editor-cli.ts add <projectId> <pageId> bg --assetId <森>
npx tsx scripts/cli/block/editor-cli.ts add <projectId> <pageId> ch \
  --characterId kaede --expressionId normal --pos C
npx tsx scripts/cli/block/editor-cli.ts add <projectId> <pageId> text \
  --speaker "楓" --body "ここが噂の魔の森か…"
npx tsx scripts/cli/block/editor-cli.ts add <projectId> <pageId> text \
  --speaker "楓" --body "引き返すなら今だけど…"
npx tsx scripts/cli/block/editor-cli.ts add <projectId> <pageId> choice \
  --text "" --options '[{"label":"進む"},{"label":"引き返す"}]'

# 5. 分岐先ページ作成
npx tsx scripts/cli/block/editor-cli.ts add-page <projectId> "魔の森 深部"
npx tsx scripts/cli/block/editor-cli.ts add-page <projectId> "村に帰還"

# 6. 整合性チェック
npx tsx scripts/cli/block/editor-cli.ts validate <projectId>
```

## スキル一覧（ノベル関連）

Claude Code に以下のスキルが登録されており、自然言語で発動する:

| スキル名 | トリガー例 | 用途 |
|---------|-----------|------|
| `edit-blocks` | 「セリフ変更」「ブロック追加」「ページ追加」 | ブロック操作全般 |
| `map` | 「マップを作って」「地形配置」 | マップ操作（RPG向け） |
| `dev-server` | 「サーバー起動」 | ローカル開発サーバー起動 |

スキルを使わず直接 CLI コマンドを指定しても同じ結果になる。
