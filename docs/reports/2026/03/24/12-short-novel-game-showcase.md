# 15秒ノベルゲーム ショーケース企画（全10本）

> Generated by Claude Opus 4.6

## 概要

kaedevn エンジンの Live2D 対応 + 65 種のエフェクトを活用した 15 秒ショートノベルゲームのショーケース企画。
**Hiyori 1 体**で全エフェクトを網羅し、エンジンの表現力を最大限に見せる。

### 目的

- ショート動画（TikTok / YouTube Shorts / X）での拡散
- kaedevn エンジンの演出力のデモンストレーション
- Live2D + エフェクトの組み合わせパターンの実例

---

## 共通仕様

| 項目 | 内容 |
|------|------|
| 時間 | 15 秒 |
| 解像度 | 1280×720（横型）or 720×1280（縦型ショート） |
| キャラ | Hiyori（Live2D 公式無料サンプル） |
| 表情操作 | afterMotionUpdate によるパラメータ上書き |
| テキスト | 1-3 行（短いほど良い） |
