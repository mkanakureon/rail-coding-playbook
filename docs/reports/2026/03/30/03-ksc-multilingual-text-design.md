# KSC スクリプト多言語テキスト設計書

> Generated by Claude Opus 4.6 — 2026年3月30日

## 背景

Phase 1-3 で UI の i18n 基盤が完成した。次の課題は **ゲーム内テキスト**（セリフ・ナレーション・選択肢）の多言語化。これは UI i18n とは異なり、**コンテンツローカライゼーション** の問題である。

UI i18n はキー固定・翻訳も開発者が管理するが、ゲームテキストはユーザーが自由に書く。翻訳もユーザー（またはAI）が行う。

## 現状のアーキテクチャ

```
ユーザーが書く     コンパイラ         ランタイム
KSC スクリプト  →  Op JSON (IR)  →  Web Engine / Native Engine
     or
ブロック JSON   →  Op JSON (IR)  →  同上
```

- `text` ブロック: `{ type: "text", body: "こんにちは", speaker: "太郎" }`
- `choice` ブロック: `{ type: "choice", options: [{ text: "はい" }, { text: "いいえ" }] }`
- テキストはブロック JSON に直接埋め込まれている

## 設計方針: ロケールオーバーレイ

### コンセプト

日本語テキストを**そのまま残し**、翻訳を**オーバーレイ**として別管理する。


### 変更対象

```typescript
// packages/compiler/src/compiler/locale-resolver.ts

export function resolveLocale(
  blocks: Block[],
  overlay: ProjectLocaleOverlay | null
): Block[] {
  if (!overlay) return blocks;

  return blocks.map(block => {
    const translation = overlay.blocks[block.id];
    if (!translation) return block;

    switch (block.type) {
      case 'text':
        return {
          ...block,
          body: translation.body ?? block.body,
          speaker: translation.speaker ?? block.speaker,
        };
      case 'choice':
        return {
          ...block,
          options: block.options.map((opt, i) => ({
            ...opt,
            text: translation.options?.[i] ?? opt.text,
          })),
        };
      case 'scroll_text':
        return {
          ...block,
          content: translation.content ?? block.content,
        };
      default:
        return block;
    }
  });
}
```

### コンパイルフロー

```
blocks → resolveLocale(blocks, overlay) → compile(resolvedBlocks) → Op JSON
```

- `locale` パラメータが未指定 or `'ja'` の場合、オーバーレイを適用しない
- プレビュー時にも locale を指定可能にする

## 変更検知とステータス管理

ソーステキストが変更された場合:
1. body の SHA-256 ハッシュを計算
2. オーバーレイの `_sourceHash` と比較
3. 不一致なら `_status: 'outdated'` に更新
4. エディタ上で黄色い警告アイコン表示

## リスク

| リスク | 対策 |
|--------|------|
| ブロック ID 変更で翻訳が迷子に | ID は安定（`{type}-{Date.now()}`）。変更時はマイグレーション |
| 大規模プロジェクトでオーバーレイ肥大化 | オーバーレイは差分のみ。未翻訳ブロックは含まない |
| AI 翻訳の品質 | `needs_review` ステータスで人間レビューを必須化 |
| ランタイム locale 切替の複雑さ | シーン切替時に限定。テキスト表示中は不可 |
