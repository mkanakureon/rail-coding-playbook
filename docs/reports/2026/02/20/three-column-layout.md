# エディタ 3カラムレイアウト実装レポート

**日付**: 2026-02-20
**ステータス**: 実装済み（動作確認中）

## 概要

PC全画面（1024px以上）で3カラムレイアウトに切り替え、ブロック編集しながらページ構成・プロパティ・プレビューを同時に見られるようにした。

## レイアウト構造

```
┌──────────────────────────────────────────────────────┐
│  Header                              [⚙設定] [保存] [▶]│
├──────────┬─┬────────────────────────┬─┬───────────────┤
│ ページ構成 │↔│ ブロックエディタ         │↔│ プロパティ      │
│          │ │                        │ │               │
│ 0 START  │ │  [START]               │ │ キャラクター:   │
│ 1 背景   │ │  [背景] bg01           │ │  太郎          │
│ 2 テキスト│ │  [テキスト] こんにちは   │ │ 表情: 通常     │
│ 3 キャラ ←──│  ← クリックで選択 ──→  │──→ 位置: C      │
│ 4 選択肢  │ │  [選択肢] 2個          │ │ 表示: ON       │
│          │ │                        │ ├───────────────┤
│ ─────── │ │  [+ ブロック追加]        │ │ プレビュー      │
│ 変数一覧  │ │                        │ │  [iframe]      │
│ hp       │ │                        │ │  P.1 [リロード] │
│ flag     │ │                        │ │               │
├──────────┴─┴────────────────────────┴─┴───────────────┤
│  ← 前  1/3  次 →  [+ ページ追加] [削除]               │
└──────────────────────────────────────────────────────┘
```

## ブレイクポイント

| 幅 | レイアウト |
|----|----------|
| < 640px | モバイル（タブ切替 + スワイプ + FAB） |
| 640px - 1023px | タブレット（タブ切替） |
| >= 1024px | 3カラム（サイドバー + 中央エディタ） |

## 左サイドバー: ページ構成 + 変数一覧

### ブロックアウトライン
- 現在ページの全ブロックをコンパクトな1行で表示
- 各行: インデックス番号 + タイプバッジ（色付き） + 要約テキスト
- クリック → 中央エディタの該当ブロックまでスクロール + 選択状態にする
- 選択中ブロックは青いハイライト + 左ボーダー

### ブロック要約テキスト

| タイプ | 要約表示 |
|--------|---------|
| start | (なし) |
| bg | アセット名 or "未選択" |
| ch | キャラ名 |
| text | 本文先頭30文字 |
| choice | "N個" |
| set_var | `varName op value` |
| if | 最初の条件式 |
| effect | エフェクト名 |
| jump | "→ P.N" |

### 変数一覧
- `getAllVariables()` で取得した全変数名を表示
- 最大高さ128pxでスクロール

## 右サイドバー: プロパティ + プレビュー

### プロパティインスペクタ（上部）
中央で選択したブロックのプロパティを編集。ブロック未選択時は「ブロックを選択してください」を表示。

| タイプ | 編集可能なプロパティ |
|--------|-------------------|
| bg | 背景画像プレビュー + セレクト |
| ch | キャラ選択 + 表情選択 + 位置(L/C/R) + 表示ON/OFF + 画像プレビュー |
| text | 話者名 + 本文(textarea) + 枠色(color picker) |
| set_var | 変数名 + 演算子 + 値 |
| effect | エフェクト種類 + 強度(1-5) + 時間(ms) |
| jump | 移動先ページ(セレクト) |
| choice | 選択肢一覧（読み取り専用。編集はブロックカードで） |
| if | 条件式表示（読み取り専用。編集はブロックカードで） |

### プレビュー（下部 200px固定）
- `ksc-demo.html` を iframe で埋め込み
- ページ番号表示 + リロードボタン
- `currentPageIndex` 変更時に自動リロード

## リサイズ機能

- 左右のサイドバーと中央カラムの間にリサイズハンドル（4px幅）
- マウスドラッグで幅を調整可能
- 左サイドバー: 180px ～ 400px（デフォルト 240px）
- 右サイドバー: 200px ～ 450px（デフォルト 280px）
- ドラッグ中はカーソルが `col-resize` に変わり、テキスト選択を無効化

## ヘッダー変更

- `isWideScreen` prop を追加
- 広い画面では ⚙️ 設定ボタンを表示
- クリックで作品設定（SettingsPanel）をモーダル表示

## 既知の制限

- 選択肢(choice)とIF文(if)のプロパティは右サイドバーでは読み取り専用。複雑な構造のため、ブロックカードを展開して編集する方式を維持
- リサイズ幅はセッション内のみ保持。リロードするとデフォルト値に戻る
- モバイル・タブレットでは従来のタブ切替式を完全に維持（変更なし）
