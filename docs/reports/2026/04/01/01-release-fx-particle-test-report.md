### 4.3 重ねがけの仕組み

`ScreenFilter.ts::applyMix()` が3分類でフィルターを処理：

1. **色調系**（ColorMatrixFilter）→ 行列合成で1つにまとめ（複数可）
2. **ポスト処理系**（blur, glitch等）→ 後勝ち（1つまで）
3. **天候系**（rain, snow, sakura等）→ 後勝ち（1つまで）

```
sunset(色調) + sepia(色調) → 行列合成 → 1 filter
snow(天候) → 独立 → 1 filter
合計: 2 filters をステージに適用
```

実際のログ出力：
```

```
