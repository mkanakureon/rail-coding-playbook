# RPGエンジン開発 経過報告書 & 次フェーズ計画

**作成日**: 2026-03-13
**作成者**: Gemini CLI
**現在の到達点**: Phase 1 (高度な表現と自動化) のコアロジック完遂

---

## 1. 完了した作業 (Milestones Achieved)

### 1.1 API & データベース連携 (Phase 1.1)
- **データ永続化**: Prisma スキーマに `Map` モデルを追加し、プロジェクトごとのマップデータ保存を可能にした。
- **CRUD API**: Hono でマップ一覧・取得・保存・削除のエンドポイントを実装完了。
- **エンジン統合**: プレビューエンジンが URL パラメータに応じて API からデータを動的にロードする仕組みを構築。

### 1.2 描画エンジンの高度化 (Phase 1.2)
- **47タイル形式オートタイル**: 斜め方向の接続まで考慮した 8-bit マスク判定ロジックを実装。
- **歩行アニメーション**: キャラクターの向きと移動状態に同期したフレームアニメーションを実装。
- **物理・表示の洗練**: キャラクター抽象化 (`MapCharacter`) と Ticker 駆動による滑らかな補間移動を実現。

---

## 2. 次のフェーズ: 次世代マップエディタの構築 (Phase 2)

これまでの「ロジック」を「直感的な操作」へ繋げるフェーズに移行する。

### 2.1 エディタ基盤の構築 (Editor Foundation)
- **API クライアントの拡張**: `apps/editor` の既存 API 通信層にマップ関連のメソッドを追加。
- **マップ一覧・作成 UI**: プロジェクト設定内に、マップを新規作成・選択できるダッシュボードを実装。

### 2.2 マップペイントエンジンの React 移植
- **Canvas/PixiJS 統合**: エディタ内でマップをリアルタイムにプレビュー・編集するための描画コンポーネントを作成。
- **タイル選択パレット**: タイルセットから使用するタイルをマウスで選択する UI。

### 2.3 イベント設定ツール
- **マップ上のノード操作**: クリックでイベントを選択し、そのプロパティ（実行シナリオ、条件等）をサイドパネルで編集。
