# 次世代物語体験：収益と美学を繋ぐ3つの「ミッシング・リンク」

> **作成日**: 2026-03-19
> **担当**: Gemini CLI

## 1. 提案の背景
AI 執筆パイプラインによる「高品質なシナリオ」と、決済システムによる「収益性」は、現在のままでは単なる「機能」の羅列に過ぎない。読者が 100 円を投じる瞬間に、それが「単なる課金」ではなく「新しい体験への入場料」であると確信させるための技術的ギミックを定義する。

## 2. 3つの技術的ギミック

### 2.1. エモーショナル・予告編 (Emotional Previewer)
- **概要**: ペイウォール到達時、AI が直前のシーンのハイライトを自動抽出し、専用の演出（BGM・エフェクト付き）でループ再生する。
- **技術的裏付け**: Stage 4 の生成データから「感情価（Valence）」の高いセリフを特定。PixiJS のフィルター機能（ぼかし、ズーム）と連携し、映画の予告編のような「続きが気になる」演出を動的に生成する。

### 2.2. 運命の分岐プレビュー (Destiny Visualization)
- **概要**: 決済画面において、読者のこれまでの選択履歴（`vars`）に基づき、この先で待ち受ける「独自の展開」を 1 行だけ AI が生成して提示する。
- **技術的裏付け**: ユーザーの変数状態を Prompt に注入し、Stage 2 で生成された複数のルート案から、そのユーザーに最適な「期待値」を抽出。決済を「物語の確定作業」へと昇華させる。

### 2.3. 共鳴型アンロック (Resonance Unlock)
- **概要**: ペイウォール画面に、リアルタイムでその作品を読んでいる他ユーザーの存在を、世界観に馴染んだエフェクト（魂の光、粒子など）として表示する。
- **技術的裏付け**: WebSocket による同期通信。他ユーザーの「アンロックの瞬間」をイベントとして受信し、画面上のパーティクルとして可視化。孤独な読書を「共同体験」へと変容させる。

## 3. 期待される効果
これらのギミックは、決済時の心理的抵抗（フリクション）を「期待感」へと反転させる。
100 円という価格設定が「安すぎる」と感じられるほどの付加価値を提供し、AI 執筆だからこそ可能な「無限のパーソナライズ」をユーザーが肌で感じる瞬間を創出する。

## 4. 結論
ミッシング・リンクとは、**「AI が書いた物語」と「人間が払う金」の間に横たわる「感情の欠如」を埋める演出**である。これらを Phase 3 以降で実装することで、kaedevn は世界で唯一の「生きている物語プラットフォーム」となる。
