# モバイル・SNS 拡散特化戦略：純粋VNエンジンの勝ち筋

> **作成日**: 2026-03-19
> **担当**: Gemini CLI
> **目的**: ツクール機能を廃しノベルゲームに特化するにあたり、ブラウザの利点（DL不要）とスマホ（縦画面・SNS）の相性を最大化するための具体的な技術・マーケティング戦略を提案する。

---

## 1. 縦画面（9:16）モードの標準化

現在の 16:9 横画面固定から、**「スマホ縦持ち（Portrait）」** を VN の第一級市民としてサポートします。

- **技術的対応**:
    - `VIRTUAL_WIDTH: 720`, `VIRTUAL_HEIGHT: 1280` のプリセットを追加。
    - 立ち絵（Character）の配置アルゴリズムを縦画面用に最適化（中央に大きく配置、腰から上のアップを基本とする）。
    - テキストウィンドウを下部に固定し、片手で親指タップしやすい UI レイアウトを標準搭載。
- **メリット**:
    - TikTok、YouTube ショート、Instagram リールの画面比率と完全一致。
    - 通勤・通学電車の「片手操作」に完全対応。

---

## 2. SNS 垂直立ち上げ：ビデオクリップ共有機能

「静止画の共有」から、**「動画（演出）の共有」** へ進化させ、SNS でのバイラルを狙います。

- **自動クリップ生成 (Share as Video)**:
    - プレイヤーが「ここ好き！」と思ったシーンでシェアボタンを押すと、直前 5〜10秒の演出（立ち絵の動き、背景エフェクト、テキスト表示）を動画化してサーバー側で生成、または WebCodecs API を利用してブラウザ内で MP4/WebP 生成。
    - 生成された動画には「続きはブラウザで即プレイ」の URL と QR コードを自動付与。
- **技術の転用**:
    - `kaede-tools` の WebP 生成ロジックや、ヘッドレスブラウザでのレンダリング（Puppeteer等）を活用し、高画質な共有用動画を作成。

---

## 3. ブラウザの強み：摩擦ゼロの「即プレイ」体験

アプリのインストールという最大の障壁を排除した「ブラウザ VN」ならではの導線を強化します。

- **シームレスな体験設計**:
    - SNS のインアプリブラウザ（Twitter や LINE 内）でリンクを踏んだ瞬間、数秒以内に 1 ページ目が表示される超高速ローディング。
    - **「お試し 1 分プレイ」**: ログイン不要、セーブ不要で、最も盛り上がるシーンから直接開始できるディープリンク機能。
- **PWA (Progressive Web App)**:
    - 気に入った作品はスマホのホーム画面にアイコンとして追加可能にし、フルスクリーンでネイティブアプリのように遊べるようにする。

---

## 4. 縦画面動画サイト（TikTok/Shorts）での拡散施策

ノベルゲームは「静止画＋テキスト」のため、動画サイトでは「動的な演出」が鍵となります。

- **実況・紹介用アセットの自動生成**:
    - 作者向けに、作品のハイライトシーンを 9:16 の縦型予告編動画として自動出力する機能をエディタに追加。
    - **AI 音声合成との連携**: テキストブロックの内容を AI 音声で読み上げ、そのまま TikTok 用の「喋るショート動画」として書き出し。
- **インタラクティブ広告形式の投稿**:
    - 動画の最後に「A か B か選んでね」という選択肢を出し、コメント欄やプロフ欄のリンクから「その選択肢の続き」を直接ブラウザで遊ばせる導線。

---

## 5. 結論：ノベル特化だからこそできる「物語の短尺化と拡散」

RPG（ツクール）は「長時間遊ぶ」体験ですが、VN は「短い物語を何度も消費・共有する」文化と親和性が高いです。

1.  **縦画面 9:16** でスマホ最適化。
2.  **ブラウザ即プレイ** でインストール不要。
3.  **動画共有機能** で TikTok/Shorts への露出を自動化。

これらを組み合わせることで、従来の「重厚長大なノベルゲーム」ではなく、**「SNS で話題になり、1 分で読み終わる、スマホ時代の新しい読書体験」** としての kaedevn ブランドを確立できます。
