# Claude Code への連携レビューおよびインポート依頼書 (2026-03-12)

## 1. コンテキスト
Gemini CLI により、AI生成パイプラインから出力される `editor-json` 等の成果物の最適化が完了した。本ドキュメントは、これを受けて Claude Code 側で行うべきインポート処理および検証作業を依頼するものである。

## 2. Gemini CLI 側で適用済みの改善点
以下の修正により、エディタへのインポート親和性が劇的に向上している。

- **IDプレフィックスの自動除去**:
  - `bg` および `ch` ブロックの ID から `$bg:`, `$ch:` プレフィックスを Stage 4 で自動的に剥離。
  - 設定ファイル (`s_02`, `s_04`) の `assetId` と完全一致した状態で出力される。
- **キャラクターIDの表記揺れ防止**:
  - システムプロンプトへのスラグ強制注入により、`garud / gard` のような綴りのブレを根絶。
- **スキーマの堅牢化**:
  - AI特有のハルシネーション（`dialogation`, `discovery` 等のキー名捏造）をパース時に自動吸収する Zod スキーマ改修を実施。
