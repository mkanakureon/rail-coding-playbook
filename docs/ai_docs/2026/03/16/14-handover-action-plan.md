## 2. PlayLayout プリセット追加

**対象**: `packages/core/src/presets/`

| プリセット名 | 用途 |
|-------------|------|
| novel-standard | ノベル標準（既存） |
| rpg-classic | RPGクラシック（既存） |
| message-top | メッセージ上部（既存） |
| message-center | メッセージ中央（既存） |
| **novel-fullscreen** | ノベル全画面（テキストウィンドウが画面全体） |
| **novel-wide** | ノベルワイド（テキストウィンドウが横幅いっぱい） |
| **rpg-battle** | RPGバトル用（ステータスパネル・コマンド配置） |
| **rpg-field** | RPGフィールド用（ミニマップ・クエストトラッカー） |
| **minimal** | ミニマル（テキストのみ、UI 最小限） |
| **cinematic** | シネマティック（黒帯 + 中央テキスト） |

## 3. UI レイアウトエディタ

**対象**: 新規 `apps/editor/src/components/panels/LayoutEditor.tsx`

ハンドオーバー資料で「UIレイアウトエディタでメニュー・ウィンドウをノーコードで再設計可能」と記載されているが**未実装**。プリセット選択のみ。

### 実装内容

- 新タブ「レイアウト」を TabNavigation に追加
- 左: 1280x720 のプレビューキャンバス（Canvas 2D で PlayLayout を描画）
- 右: 選択中の要素のプロパティパネル（x, y, width, height, visible, opacity）
- 要素クリックで選択、プロパティ変更で即プレビュー更新
- プリセット選択ドロップダウン（ベースを選んでカスタマイズ）
- 保存: project.data.playLayout に JSON 保存
