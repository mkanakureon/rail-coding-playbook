# PR確認レポート：RPGエンジン・UIレイアウト統合

**作成日**: 2026-03-14
**ステータス**: ステージング中 / レビュー完了
**対象ブランチ**: `main`

## 1. 概要
本PRは、kaedevn をノベルゲーム専用エンジンから、RPG/ADV ハイブリッド制作ツールへと進化させる大規模な機能統合です。

## 2. 主要な成果
- **マップレンダラー**: 複数レイヤー、8方向オートタイル、スムーズな補間移動を実装。
- **UIレイアウトエンジン**: `@layout_set` 命令により、メッセージウィンドウの位置やデザインを動的に切り替え可能。
- **API/DB連携**: Prisma への `Map` モデル追加と、Hono API への `/maps` エンドポイント追加。
- **テスト自動化**: RPGフロー、マップ機能、UIレイアウトの3つの E2E テストを完備。

## 3. マイグレーションに関する警告（最優先事項）
ユーザーからのメモに基づき、以下の点を確認しました。
- **マイグレーション不足**: ステージングされている `schema.prisma` の `Map` モデル追加に対応するマイグレーションファイルがまだ作成されていません。
- **デプロイ未完了の可能性**: 02-24に発生した `assets.slug` のエラーは、`20260225_add_missing_columns` で解消されるはずですが、これが本番に適用されているか再確認が必要です。
- **推奨アクション**: コミット前に `npx prisma migrate dev --name add_map_model` を実行し、マイグレーションファイルを生成してください。
