# 作者成功戦略：最初の100人を熱狂させるための施策案

> **作成日**: 2026-03-23
> **目標**: kaedevn を「一番簡単に、一番反応がもらえるエンジン」として確立し、100人のアクティブな作者を確保する。

---

## 1. 「30分以内の成功体験」の提供（摩擦ゼロ化）

作者候補がサイトを訪れてから、最初の作品を公開するまでの心理的障壁を徹底的に排除する。

- **AI 執筆アシスタント (GenAI 連携)**:
    - 3/18 に導入した Imagen 4.0 連携を拡張。プロットを入力するだけで「ふさわしい背景・キャラ・初期スクリプト」を自動生成する「AI 下書き機能」。
- **素材探しの旅を終わらせる**:
    - 公式アセット（タイルセット、立ち絵）をタグ検索で即座に配置可能にする。
- **スマホ編集の完全開放**:
    - 3/20 の縦画面対応を活かし、PC を開かずにスマホだけで「セリフの修正」「シーンの追加」ができる UI の磨き込み。

## 2. 「反応の可視化」による執筆意欲の維持（承認欲求）

作者が最も恐怖するのは「誰にも読まれないこと」である。システムが読者を連れてくる仕組みを構築する。

- **「注目の新人」アルゴリズム**:
    - 過去の実績に関わらず、新規公開作品をトップページの特等席に 24 時間固定表示し、最初の 100 プレイを保証する。
- **リアルタイム・エンゲージメント通知**:
    - 「あなたの作品が今、〇〇人に読まれています」「〇〇ページ目でいいねされました」というプッシュ通知。
- **作者専用アナリティクス（読了率ヒートマップ）**:
    - 「読者がどこで離脱したか」をページ単位で可視化。作者に「次はここを直そう」という改善の楽しみ（ゲーム性）を与える。

## 3. 「執筆の孤独」を解消する（コミュニティ）

一人で作り続ける限界を、コミュニティと運営のサポートで突破する。

- **Discord 連動型の開発**:
    - 作者限定の Discord チャンネルを開放。バグ報告に開発者が即レスする「神対応」を 100 人規模までは徹底し、ファン化させる。
- **共同制作モードの布石**:
    - 「シナリオ担当」と「イラスト担当」が同じプロジェクトを編集できる権限管理機能。
- **定期的な「制作祭（ジャム）」の開催**:
    - 「週末で 5 分の作品を作る」といった緩いテーマのイベントを公式で開催し、締め切りによる強制力と連帯感を作る。

## 4. 「将来の収益化」という夢（インセンティブ）

3/19 の「ポイント課金システム（モック）」を早期に実機能化し、作者に「ここで書けば稼げる可能性がある」という期待感を持たせる。

- **投げ銭・ペイウォール機能の早期開放**:
    - 100 人の作者の中から、特に熱量の高い数名に限定して先行収益化テストを開始する。
- **エクスポート機能の優遇**:
    - 定着している作者（1 万文字以上執筆など）に対し、macOS/Android エクスポート機能を無料で開放し、自身のブランド構築を支援する。
