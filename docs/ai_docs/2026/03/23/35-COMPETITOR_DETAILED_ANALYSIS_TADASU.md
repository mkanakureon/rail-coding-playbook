# 競合調査：ただのべる (Tadasu Novel) と kaedevn の戦略的差異

> **分析日**: 2026-03-23
> **対象**: https://www.tadasunovel.com/

---

## 1. 競合の強み（学ぶべき点）
- **究極のシンプルさ**: フォーム入力だけで物語ができる。専門用語を一切排除している。
- **ブラウザ完結**: インストール不要。
- **先行者利益**: 既に 800 件以上の作品という「資産」がある。

## 2. 競合の弱点（攻めるべき点）
- **演出の貧弱さ**: 2010 年代のブラウザゲームのような静的な UI。
- **モバイル UX の欠如**: スマホで動くが、スマホに最適化（縦画面でのフル体験）されていない。
- **拡散機能の欠如**: SNS で「映える」仕組み（動画、リッチな OGP）がない。
- **データフィードバックの欠如**: 作者が読者の反応を詳しく知る術がない。

## 3. kaedevn の勝利条件
「ただのべる」より **10 倍簡単** に作り始められ、**100 倍豪華** な演出し、**1000 倍広まる** プラットフォームになる。

### A. 乗り換え促進 (Import Strategy)
既存の「ただのべる」作者が、過去作を kaedevn に移すだけで「最新のスマホ対応」「動画生成」「アナリティクス」が手に入るという価値を提示。

### B. 動画マーケティングの独占
「動画を作れるのは kaedevn だけ」というポジションを確立し、TikTok/YouTube Shorts からの流入を独占する。

### C. コミュニティの「鮮度」
停滞している競合に対し、「毎日機能が増える」「Discord で開発者に会える」という圧倒的な鮮度の差で作者を引き付ける。

---

## 4. 総評
「ただのべる」の 839 作品は、kaedevn にとっての「見込み顧客リスト」である。彼らが抱えている「もっと綺麗にしたい」「もっと広めたい」という不満を、私たちの最新実装（縦画面、動画、スタンプ）で解決することで、市場の主導権を握ることができる。
