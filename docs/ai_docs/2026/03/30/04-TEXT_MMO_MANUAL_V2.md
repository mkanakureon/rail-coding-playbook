## 2. 核心技術：マルチプレイ同期システム

### ① CommandSync (命令同期プロトコル)
- **機能**: プレイヤーが発行した KSC 命令（移動、発言、バトルコマンド）を、他の全プレイヤーのランタイムへ低遅延でブロードキャストする。
- **監査所見 (Layer 1)**: 通信パケットを OpCode 単位に圧縮することで、モバイル回線環境下でも O(1) の低負荷な同期を実現している。

### ② Cloud-Save v2 ＆ 日次バックアップ
- **機能**: 全プレイヤーのステータス、Entity Graph の動的変化を Azure DB / PostgreSQL へリアルタイム保存。
- **信頼性**: `pg_dump` 自動スケジューラと日次メール通知により、運営型ゲームに不可欠な「データの整合性」と「復旧可能性」を保証。

### ③ Entity Graph による動的世界構築
- **機能**: マップ接続、エンカウント設定、トループ（敵グループ）の動的生成を GUI から即座に変更可能。
- **特徴**: 開発者がエディタでマップを 1 マス書き換えるだけで、全プレイヤーの「現在の世界」が同期更新される。
