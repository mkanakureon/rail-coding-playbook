# YouTubeを活用した広報・マーケティング戦略：動画によるOSSの価値証明

## 1. 概要
Playwrightにより自動生成されたデモ動画をYouTubeに公開することで、kaedevnの認知度向上、信頼性獲得、およびユーザー流入を加速させる戦略を分析しました。

## 2. YouTube投稿の3つの戦略的意義

### ① 視覚的な「クイック・デモ」
- 言葉での説明が難しい「ブロックをクリックした瞬間にプレビューが変わる（CPC機能）」といったUXを、10秒の映像で直感的に理解させることができます。

### ② AI駆動開発（AI-Native）のブランディング
- 「このデモ動画自体が、AIエージェントによる自動操作で作成されたものである」という事実は、kaedevnが次世代の技術スタックであることを強く印象付けます。

### ③ ユーザーの「成功体験」の予習
- 投稿された動画は、新規ユーザーにとっての「動画マニュアル」になります。動画を見ることで「自分でも作れそうだ」という自信を与え、OSSのクローン率（利用率）を高めます。

## 3. 効果的な動画構成案（PVスタイル）
1. **イントロ (5秒)**: 綺麗な背景とロゴ。
2. **コア機能 (30秒)**: エディタでの直感的な操作シーン。
3. **プレイ画面 (30秒)**: PC-98エフェクトやタイムラインによる滑らかな演出。
4. **アウトロ (5秒)**: GitHubのリポジトリURLと、"Free & OSS" の文字。

## 4. 運用上の注意
- **BGM/SEの重要性**: Playwright録画には音声が含まれないことが多いため、編集段階で無料のフリー音源を合成し、視聴維持率を高めます。
- **定期的な更新**: UIが大きく変わった際は、スクリプトを再実行して新しい動画を投稿し直すことで、プロジェクトが活発であることを示し続けます。

## 結論
YouTubeは単なる動画置き場ではなく、全世界のクリエイターに向けた「窓口」です。自動生成される「嘘偽りのない完璧な操作動画」を継続的に発信することで、kaedevnをノベルゲーム制作の第一選択肢へと押し上げることが可能です。
