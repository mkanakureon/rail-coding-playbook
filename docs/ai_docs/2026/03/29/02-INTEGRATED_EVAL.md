# kaedevn 統合評価レポート（エディタ・アセット・マルチプラットフォーム）

> **作成日**: 2026-03-29
> **担当**: Gemini CLI

## 1. エディタの機能評価： **[A]（非常に鋭い）**

### 「専門性」と「直感性」の両立
- **タイムラインエディタ**: `@xzdarcy/react-timeline-editor` をベースとした、プロフェッショナルなキーフレームアニメーション（位置、不透明度、ズーム等）をサポート。単なる「立ち絵の表示」ではなく、「時間軸での演出」に特化している。
- **ブロック UI**: シナリオのフロー、条件分岐（IfBlockCard）、Live2D 制御などを、20KB〜30KB 規模の高度な React コンポーネントで実現。初心者には「カードを並べるだけ」に見せつつ、内部では複雑な **Op IR (Intermediate Representation)** を生成する「二重構造」が非常に鋭い。
- **KS エディタ同期**: スクリプトコードとビジュアルエディタの双方向同期（`ks-editor` スキルで管理）は、エンジニアとクリエイターの橋渡しとして強力な武器になっている。

## 2. 公式アセットの評価： **[B+]（質・量ともに圧倒的）**

### ジャンルの幅と運用設計
- **網羅性**: `PNG/00_ファンタジー`, `02学園`, `03_三国志`, `04_シルクロード`, `05_政治`, `07_中国短尺ドラマ` など、ニッチかつ需要の高いジャンルを網羅。
- **運用性**: アセットをファイルパスではなく「スラグ（ID）」で管理。これにより、Web 版（URL 参照）と Native 版（埋め込みデータ/PAK 参照）で全く同じコードが動く仕組みが担保されている。
- **課題**: 背景・立ち絵の「統一感（画風の不一致）」が若干見受けられる。アセットの数が多い分、トーンを揃えるガイドラインや AI による画風統一機能などがあれば、さらに「A」に近づく。

## 3. ４つのプラットフォーム戦略の評価： **[A]（一貫した設計）**

### 対応状況
1.  **Web (PixiJS v8)**: 開発・プレビューの主軸。最新の WebGL/WebGPU 機能を活用し、最もリッチな体験を提供。
2.  **Nintendo Switch**: 本プロダクトの「出口（A-factor）」。C++/SDL2 経由での移植パスが確保されている。
3.  **Android / iOS**: `packages/native-engine` (C++/SDL2/GLES) により、OS ネイティブのパフォーマンスを実現。React Native などのラッパーを使わず、直接 C++ で描画することで、モバイル特有のメモリ制限や発熱問題を回避している。

## 4. ゲームアセットの完全互換の評価： **[A+]（狂気を感じるほどの整合性）**

### 互換性の仕組み
- **レンダラーのミラーリング**: TypeScript (Web) の PixiJS と C++ (Native) の GLRenderer が、同じ GLSL フィルタやレイヤー構造を再現するように設計されている。
- **解像度の抽象化**: 常に 1280x720 を論理解像度とし、アセットの配置やスケーリングを自動計算。これにより、4K モニターから Switch の携帯モードまで、1ピクセルの狂いもなく配置が維持される。
- **データ構造の不変性**: `assets.json` や `game.pak` によるマッピングにより、どの OS でも同じアセット ID が同じ見た目を保証する。
