## 3. パーティクル演出の評価 (`SakuraFilter.ts` 他)

### 技術的ハイライト（桜パーティクル）
*   **3 層視差 (Parallax):** `petalLayer` を異なるスケールと速度で 3 重に重ねることで、2D 画面に圧倒的な奥行きを与えている。
*   **高度な GLSL 演算:**
    *   `Horizontal drift`: `sin` 波を用いたランダムな揺らぎ。
    *   `Rotation`: 個々の花びらに異なる回転速度を付与。
    *   `Color Post-process`: パーティクルの強度（`uIntensity`）に連動して、画面全体にわずかな暖色ティントをかける演出的な配慮。
*   **風 (uWind) の導入:** パラメータ一つで「風向き」を変えられる設計は、シナリオの状況（海辺、嵐など）に合わせた柔軟な演出を可能にしている。

### 評価
「降らせるだけ」のパーティクルではなく、背景と馴染ませるための「空気感の演出」まで踏み込んで実装されており、メメントモリ級の描画リッチさを Web 環境で実現できている。
