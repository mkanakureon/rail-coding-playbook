# 実験報告：禁止ワード（NGワード）の削除による表現の変化と評価

## 1. 実験の目的
リカバリ戦略 (28番) で設定した「禁止ワード（瘴気、侵食、捕食者等）」が、物語の表現力（ドラマ性）にどのような影響を与えているかを検証する。
「負の制約（禁止）」を解除し、「正の誘導（キーワード、リファレンス）」のみでトーンが維持できるかを確認する。

## 2. 比較分析

### A. 禁止ワードあり (前回生成: 220306)
- **トーン**: 水彩画のような透明感、穏やか、極めて安全。
- **特徴**: 描写が「美しさ」に寄りすぎており、村の危機（泉が枯れる）という重大な事態に対しても、どこか他人事のような、あるいは幻想的な手触りが強かった。
- **評価**: 低年齢層向けとしては完璧だが、ドラマとしての「切実さ」に欠ける。

### B. 禁止ワードなし (今回生成: 221836)
- **トーン**: じりじりとした残暑、涸れ果てた岩肌、村人たちの重い不安。
- **特徴**: 「絶望」や「冷たい」といった言葉の自由度が増したことで、村が直面している危機のリアリティが向上した。一方で、リファレンス（ドラクエIV等）の効果により、ダークファンタジー化（グロテスクな表現等）は発生しなかった。
- **評価**: **極めて良好**。王道冒険譚としての「試練」と、それを乗り越える「小さな勇気」の対比が、よりドラマチックに描かれている。

## 3. 結論：ガードレールの再定義
- **負の制約（NGワード）**: 物語の「感情の幅（ダイナミックレンジ）」を狭めてしまうリスクがある。特に「絶望」を完全に禁じると「希望」の描写も弱まる。
- **正の誘導（キーワード・リファレンス）**: Gemini 2.5 系列において、トーンの維持には「ドラクエIV」「風ノ旅ビト」といった具体的な世界観リファレンスと、「素朴さ」「温もり」といった正のキーワードだけで十分な強制力を持つことが判明した。

## 4. 今後の方針
- **禁止ワードリストの廃止**: プロンプトからは削除する。
- **正のトーンガイドの継続**: `RECOVERY_TONE_GUIDE` の「トーン・キーワード・リファレンス」を今後も全ステージのシステムプロンプトに注入し続ける。
