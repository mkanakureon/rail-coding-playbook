# kaedevn ツクール型エディタ：収益予測（楽観＋Switch 参入シナリオ）

> **作成日**: 2026-03-18
> **担当**: Gemini CLI
> **前提**: Month 3 に「Nintendo Switch 書き出し・出版サポート」を開始。中堅以上の有力クリエイター層が流入し、作品の質とプラットフォームの信頼性が爆発的に向上するシナリオ。

## 1. 収益モデルとコンバージョンの前提（Switch 追加版）

従来の 4 つの柱に、強力な 5 番目の柱を追加します。

1.  **💾 容量課金 (300円/月)**：全作者の 5%
2.  **🚀 4プラットフォーム書き出し (200円/月)**：全作者の 10%
3.  **🛍️ 販売手数料 (流通の10%)**：ARPU 10円〜20円
4.  **📢 広告収益 (250RPM)**：1人10PV想定
5.  **🎮 Switch 出版サポート（新規！）**
    *   **出版事務手数料**: **50,000円 / 本**（eショップ登録、レーティング申請サポート代行等）。
    *   **レベニューシェア**: **売上の 10%**（Nintendoの手数料30%を除いた残りから、プラットフォーム維持費として徴収）。
    *   **予測**: 有力作品が Month 5 あたりから順次 Switch でリリース開始されると想定。

## 3. Switch 参入がもたらす戦略的パラダイムシフト

### 3.1. 圧倒的な「ブランド信頼性」の獲得
「Switch に出せるエンジン」という肩書きは、クリエイターにとって最大の信頼の証です。これにより、これまで「無料ツールは不安」と躊躇していた商業・プロ志向の作家が一気に参入し、作品のクオリティが底上げされます。

### 3.2. 利益率の爆発（LTVの増大）
Web広告は薄利多売ですが、Switch でヒット作が出た場合の「10% レベニューシェア」は、1作品だけで数百万円〜数千万円の収益をプラットフォームにもたらす可能性があります。
*   5万円の事務手数料は、サポート工数の「原価」として機能し赤字を防ぎます。
*   売上10%は、**「プラットフォームがクリエイターの成功を全力で応援する」** というインセンティブになり、強力なパートナーシップが築けます。

### 3.3. 読者（MAU）への相乗効果
「Switch で人気のあのゲームの『前日譚』や『外伝』が、kaedevn の Web版で無料で今すぐ遊べる！」という導線を作ることで、コンソールから Web へ、Web からコンソールへとプレイヤーを回遊させる「メディアミックスのハブ」になれます。
