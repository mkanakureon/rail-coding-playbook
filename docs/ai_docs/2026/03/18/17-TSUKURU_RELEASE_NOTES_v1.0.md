## 2. 主要な機能ハイライト

### 2.1. 完璧なツクール互換 UI
*   **ダークテーマ**: 本家ツクール MZ を忠実に再現した `#1a1a2e` ベースの配色。
*   **直感的な操作**: ツールバーでの「マップ/イベント」モード切替、右クリックスポイトによるタイル取得。
*   **高度な DB エディタ**: タイルセットの通行判定（○/×/☆）をグリッド上でクリック切替。トループのドラッグ＆ドロップ配置。

### 2.2. 強力なイベントスクリプトエディタ
*   **◆（インデント）表記**: ツクール特有の「◆ 文章の表示」といった視覚的なコマンドリストを実装。
*   **ネスト（入れ子）の完全サポート**: `CommandPath` ユーティリティによる再帰的データ操作により、条件分岐や選択肢の中に無制限にコマンドを配置可能。
*   **WYSIWYG プレビュー**: 16:9 のゲーム画面を模した領域で、文章の表示位置や改行をリアルタイムに確認可能。
*   **DB 連携ドロップダウン**: 場所移動やバトル処理における、手入力を排した安全な ID 選択 UI。

### 2.3. ブラウザで動くランタイムエンジン
*   **非同期インタプリタ**: Promise ベースで実装された `EventInterpreter` により、クリック待ちやウェイトといった RPG 特有の「時間制御」を React 上で矛盾なく動作。
*   **F9 リアルタイムデバッガ**: テストプレイ中に起動し、変数やスイッチの状態を監視・書き換え可能。

## 3. アーキテクチャと技術的負債の排除

*   **Zustand による一元管理**: `useTsukuruStore` によって、プロジェクトデータと UI 状態（選択中タブ、描画ツール等）を完全に分離。
*   **徹底したコンポーネント分割**: GameDbPanel を 13 個のタブファイルに分割し、`DbFieldHelpers` で共通フォームを再利用することで、コードの肥大化を防止。
*   **法的リスクの回避**: 「ツクール」という名称や公式アセット（画像・アイコン・コード）を一切使用せず、商用利用可能なフリー素材と Tailwind CSS による独自のスタイリングで構築。

## 4. 今後のロードマップ (v1.1 以降)

本リリースにより「ツクール作者が違和感なく使える基盤」が完成しました。今後は以下の Web ネイティブな拡張機能を予定しています。

2.  **ビジュアルスクリプト**: ノードを線で繋ぐ、高度なイベント構築モード。
3.  **複数人での共同編集**: Figma のようなリアルタイム同期マッピング機能。
