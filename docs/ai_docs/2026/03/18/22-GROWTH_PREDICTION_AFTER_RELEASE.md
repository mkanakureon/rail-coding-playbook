# ツクール型エディタ リリース後の成長予測（1年後）

> **作成日**: 2026-03-18
> **担当**: Gemini CLI
> **前提**: kaedevn ツクール型エディタ（v1.0）および次世代UIエディタの実装完了・一般公開を起点（Month 0）とする。

本ドキュメントは、ツクール作者という巨大な既存コミュニティに対し、学習コストゼロで「Web/スマホ対応」「UI自由化」という価値を提供した場合の、プラットフォーム全体の成長（作者数、読者/プレイヤー数、作品数）を3つのシナリオで予測します。

---

## 1. 成長のドライバー（なぜ伸びるのか）

*   **作者獲得のフック**: 「MZと同じ操作感」＋「スマホで遊べる」＋「プラグインなしでUIが作れる」。ツクール作者の3大ペイン（苦痛）を完全に解消しているため、乗り換えのハードルが極めて低い。
*   **読者獲得のフック**: `.exe` ファイルのダウンロードやアプリのインストールが不要。X（旧Twitter）のURLから1タップでブラウザ（スマホ）上でRPGが起動するため、バイラル（拡散）係数が圧倒的に高い。

---

## 2. 成長予測シナリオ（リリースから1年後）

### 📊 【普通】シナリオ（Base Case）
*口コミとSNSでの地道な広がりにより、着実なコアコミュニティが形成される状態。*

*   **作者（Creators）**: **3,000 〜 5,000 人**
    *   初期は新しいツール好きや、スマホ向けにRPGを作りたかった層が流入。月に数百人ペースで定着。
*   **作品（Works）**: **1,500 〜 2,500 本**
    *   短編RPGや、既存のツクール作品の移植版が多く公開される。エタる（未完成）作品も多いが、遊べる状態のものが順調に蓄積。
*   **読者/プレイヤー（Players）**: **50,000 〜 100,000 人/月**
    *   作者がSNSで共有したURL経由で、フォロワーや友人がスマホでカジュアルにプレイする。

### 🚀 【楽観】シナリオ（Optimistic / Best Case）
*有名クリエイターの参入や、バズる（大ヒットする）名作RPGがプラットフォームから誕生し、爆発的に普及する状態。*

*   **作者（Creators）**: **20,000 〜 50,000 人**
    *   「UIが自由に作れる」という噂が広まり、Unity等に挫折した層や、ノベルゲーム作者までもがRPG制作に参入。
*   **作品（Works）**: **10,000 〜 20,000 本**
    *   UIレイアウトエディタを駆使した「見た目がツクールっぽくない」スタイリッシュな商業レベルのインディーゲームが多数公開される。
*   **読者/プレイヤー（Players）**: **500,000 〜 1,000,000 人/月**
    *   YouTuberやVTuberが「ブラウザで今すぐ遊べるフリゲ」として実況配信を行い、視聴者がリンクから一斉に押し寄せる。サーバーのオートスケールが必須な状態。

### 📉 【厳し目】シナリオ（Pessimistic / Worst Case）
*ツクールからの乗り換えコスト（プラグイン資産の放棄など）が予想以上に重く、アーリーアダプター止まりになる状態。*

*   **作者（Creators）**: **500 〜 1,000 人**
    *   「やっぱり長年使い込んだMZのプラグインが使えないとキツイ」と、既存ユーザーの定着率が低い。
*   **作品（Works）**: **200 〜 400 本**
    *   技術デモやチュートリアルレベルの小品が多く、長編の大作RPGがなかなか生まれない。
*   **読者/プレイヤー（Players）**: **5,000 〜 10,000 人/月**
    *   コミュニティ内での相互プレイにとどまり、外部の一般ゲーマー層までリーチしない。

---

## 3. 【楽観】シナリオへ引き上げるための重要施策

「普通」や「厳し目」のシナリオを回避し、爆発的成長（楽観シナリオ）の軌道に乗せるためには、リリース後の以下の施策が鍵を握ります。

1.  **「脱・ツクールっぽさ」の公式ショーケース**
    *   UIレイアウトエディタを極限まで活用し、「えっ、これツクール風のエディタで作ったの？」と驚かれるような、UIがスタイリッシュな公式サンプルゲーム（短編）をリリースと同時に公開する。
2.  **スマホ向け「縦持ちRPG」テンプレートの提供**
    *   最初から画面が縦長（9:16）で、片手で操作しやすいUIが設定済みのプロジェクトテンプレートを用意する。スマホゲー世代のクリエイターとプレイヤーを一気に取り込む。
3.  **アセットストア（素材共有）の拡充**
    *   作者がツクールMZから乗り換えられない最大の理由は「素材（特にJSプラグイン）の互換性」です。プラグイン不要で高度なシステムが組めることの啓蒙と、フリーの画像・音楽素材をエディタ内で直接検索・インポートできる機能を強化する。
