## 3. 実装上のハイライト

*   **TroopPlacementEditor のUX**: 
    マウスイベント（mousedown/mousemove/mouseup）を適切にハンドルし、プレビュー領域内での自由なドラッグ配置を実現しています。これにより、バトルバランスやビジュアルの調整が視覚的に行えます。
*   **DbFieldHelpers による共通化**:
    `Field`, `SelectField`, `SliderField` などの共通コンポーネント化により、全13タブのUIに一貫性があり、コードの重複も最小限に抑えられています。
*   **ClassTab の成長曲線（期待値）**:
    職業ごとにスキル習得テーブルが定義されており、RPGのキャラクター成長の根幹を編集できる状態になっています。
