# グロースハック向け：前倒し実装機能一覧（Month 3 成功の鍵）

> **作成日**: 2026-03-18
> **担当**: Gemini CLI
> **目的**: Month 3 のゲームジャム成功とバイラル拡散（SNSでのバズ）を確実にするため、マーケティング施策をシステム面から自動化・後押しする機能要件を洗い出し、優先実装リストとして定義する。

---

## 1. 最優先（ローンチ前〜Month 1 に必須）
これらの機能がないと、せっかくの「スマホで遊べる」という強みがSNSで拡散されません。

### 🥇 1. 動的 OGP（サムネイル）画像生成システム
*   **目的**: X（旧Twitter）等で作品URLがシェアされた際のクリック率（CTR）を最大化する。
*   **実装内容**: 
    *   Next.js の動的 OGP 生成（`@vercel/og` 等）を利用。
    *   単なるロゴではなく、**ゲームのタイトル、作者名、実際のマップのスクリーンショット（またはイメージ画像）を合成した OGP 画像**を自動生成して返す API の構築。
    *   プレイ画面の `<meta>` タグへの動的埋め込み。

### 🥇 2. プロジェクト「テンプレート（プリセット）」機能
*   **目的**: 「スマホ向けUIを作るのが面倒」という初期離脱を防ぐ。
*   **実装内容**:
    *   新規プロジェクト作成時、単なる空のデータではなく「スマホ縦持ち RPG（9:16）」「クラシック RPG（16:9）」といった**テンプレートを選択できる UI**（マイページ側）。
    *   裏側では、あらかじめ調整済みの `PlayLayout` データと、初期マップ、基本的なシステム設定が含まれた JSON をコピーしてプロジェクトを初期化するロジック。

---

## 2. 優先（Month 2〜ジャム開催までに必須）
ゲームジャム（お祭り）の熱量を高め、プレイヤー（読者）を巻き込むための機能です。

### 🥈 3. ポータルサイト（Next.js）のイベント・タグ機能
*   **目的**: ジャム参加作品を一覧化し、お祭り感を演出する。
*   **実装内容**:
    *   作品公開時に「#第1回スマホRPG制作祭」などの**公式タグを選択・ロックできる機能**。
    *   マイページ（ポータル）のトップに、特定のタグがついた作品だけを並べる「特設セクション」のコンポーネント実装。

### 🥈 4. プレイヤー投票（いいね / 評価）UI の強化
*   **目的**: プレイヤーの熱量を可視化し、作者の承認欲求（モチベーション）を満たす。
*   **実装内容**:
    *   既存の Prisma スキーマにある `likeCount` を活用。
    *   ゲームプレイ終了時（または画面の隅）に、**ログイン不要（または簡易ログイン）で「いいね（👍）」を押せる目立つボタン**の実装。
    *   ポータルサイトに「急上昇（直近のいいね数）」ランキングの実装。

---

## 3. 中長期（ジャム後のリテンション向上）
ジャムで集まった作者を定着させるための機能です。

### 🥉 5. プレイ統計（アナリティクス）の作者向けダッシュボード
*   **目的**: 作者に「自分のゲームがどれだけ遊ばれているか」という強烈なフィードバックを与える。
*   **実装内容**:
    *   マイページのプロジェクト管理画面に、「累計プレイ回数」「ユニークプレイヤー数」「クリア率（特定の変数・イベント到達率）」をグラフ表示する機能。

### 🥉 6. クリエイター収益化とサブスクリプション・エコシステム
*   **目的**: 素材探しのペイン解消と、作者が「作って稼げる」超クリエイターファーストな経済圏の確立。
*   **実装内容**:
    *   **公式アセットの即時利用**: 既に用意されている 500 枚の公式画像アセットを無料で使い放題にする。`ResourceManager.tsx` 内の「公式ライブラリから探す」タブからワンクリックで自プロジェクトへインポート可能。
    *   **毎月のゲームジャム定例化**: 単発のお祭りではなく「毎月開催」をシステム化し、常に新しい作品が生まれ、遊ばれる熱量を維持する。
    *   **上位作品の有料販売と超低手数料**: 上位作品や人気作を作者自身が有料販売できる仕組みを実装。販売手数料は決済インフラ費（Stripe等）とシステム維持費で赤字にならない限界の **一律 8% 〜 10%**（業界最安水準・作者還元率90%以上）に設定する。
    *   **ストレージ容量課金（フリーミアム）**: プロジェクトの保存容量 **100MBまでは完全無料**。本格的な長編RPGなどで **100MBを超える場合は 月額 300円** の月額課金プランとする。
    *   **4プラットフォーム書き出し（月額 200円）**: 「ブラウザで動くものはネイティブでも動く」という技術検証済みの強みを活かし、**スマホ（iOS / Android）およびPC（Windows / Mac）の4プラットフォームへのアプリ書き出し機能**を実装。この強力な書き出し機能は **月額 200円** のオプションとして提供する。

---

## 4. 開発への組み込み方針
*   **動的 OGP** と **テンプレート機能** は、エディタの完成後（UIレイアウトエディタ等と並行して）、Next.js 側の最優先タスクとしてバックログに追加します。
*   これらは「ゲームエンジン」の機能ではありませんが、「プラットフォームを大ヒットさせる」ためにはエンジン開発と同等以上の価値を持つ重要機能です。
