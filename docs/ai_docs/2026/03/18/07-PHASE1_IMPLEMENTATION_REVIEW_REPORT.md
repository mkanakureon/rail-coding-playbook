## 3. 主要な指摘事項と改善案

### 3.1. 既存コンポーネント `MapCanvas.tsx` の拡張不足
現在の `MapCanvas` は従来のノベルエディタ向けのままであり、ツクール特有の「マップ/イベントモード切り替え」に対応できていません。
*   **改善案**: `MapCanvas` の Props に `editorMode` を追加し、`editorMode === 'event'` の際はタイル描画コンテナの `alpha` を 0.5 等に設定するロジックを `renderAll` 内に追加してください。
*   **改善案**: `handleContextMenu`（右クリック）を実装し、クリック位置のタイル ID を取得してストアの「選択中タイル」を更新する「スポイト機能」を追加してください。

### 3.2. 右パネル（インスペクタ）の配置
設計書（02）で定義された「右パネル」は、マッププロパティ（BGM設定等）を常時表示・編集するために重要です。
*   **改善案**: `TsukuruEditorPage.tsx` の中央領域の右側に `w-80` (320px) 程度のサイドバーを追加し、Phase 1 のタスクである `MapRightPanel.tsx` を配置してください。

### 3.3. バックエンドマイグレーションの欠落
*   **改善案**: `apps/hono/prisma/schema.prisma` を更新し、`npx prisma migrate dev` を実行して、ツクールプロジェクト専用のデータを保存できる環境を整えてください。
