# 05-AGENTS_RAG_BENEFITS_ANALYSIS

## 1. コンテキスト節約のメリット
ハイブリッド検索の導入により、エージェントは必要な情報の核心部分のみを取得できるようになり、コンテキスト窓の 90% 以上を節約可能となる。

## 2. 検索精度の向上
- **キーワード検索**: 固有名詞やバージョン番号を確実にキャッチ。
- **ベクトル検索**: 「あの機能の設計思想」といった抽象的な問いに対応。

## 3. エージェントの自律性
エージェントが自ら `search_knowledge_base` を呼び出すことで、人間がパスを教える手間が省け、自律的な問題解決能力が向上する。
