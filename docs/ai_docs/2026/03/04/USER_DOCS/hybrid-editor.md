# ハイブリッド・エディタの仕組み

## 1. 2つの編集モード
### 📝 Script (KS) モード
プロ仕様のコードエディタで直接「Kaede Script」を記述。大量のセリフ入力に最適。

### 🧱 エディタ（ブロック）モード
演出や設定を視覚的なブロックとして管理。ドラッグ＆ドロップで直感的に組み立て。

## 2. 双方向同期
どちらのモードで編集しても、もう一方に即座に反映されます。
1. Scriptで書き、[Save & Sync] を押す。
2. ブロック画面に自動反映。
3. ブロックで微調整すると、Scriptも自動更新。

## 3. クリエイターの意図を保護
スクリプト内に書き込んだ `; メモ書き` コメントは、同期を経ても消えずに保持されます。
