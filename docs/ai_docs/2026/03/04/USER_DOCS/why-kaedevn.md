# kaedevnが選ばれる理由

kaedevn（かえで）は、**「物語を書きたい」という情熱を、一切の妥協なしに形にする**ために設計されたビジュアルノベル制作プラットフォームです。

## 1. 「書く」と「作る」の完璧な融合（ハイブリッド・エディタ）

多くの制作ツールは、「初心者向けのGUI（ブロック式）」か「上級者向けのスクリプト」のどちらかに偏っています。kaedevnはその境界をなくしました。

- **Script（KS）モード**: タイピングが得意な方は、プロの作家のようにテキストで一気に書き進められます。
- **エディタ（ブロック）モード**: 複雑な演出やフラグ分岐は、視覚的なブロックで直感的に組み立てられます。
- **双方向同期**: テキストで書いた内容は即座にブロックに、ブロックで調整した内容は即座にテキストに反映されます。

## 2. 豊富な「すぐに使える」素材アセット

真っ白な画面から始める必要はありません。kaedevnには、クオリティの高いプリセット素材が標準搭載されています。

- **ファンタジー世界**: 冒険の舞台となる背景や立ち絵が揃っています。
- **現代・学園**: 日常の物語をすぐに描き出せます。
- **公式アセット**: プロ仕様の素材を自由にプロジェクトに組み込めます。

## 3. 作った瞬間、そこが「世界」への入り口

kaedevnで作品を保存すれば、それはもう公開準備が整っています。

- **ブラウザで遊ぶ**: URLを共有するだけで、PCでもスマホでもプレイ可能です。
- **ネイティブアプリ化**: 既にAndroid版やPC版（Mac/Linux）での動作実績があります。
- **コンソールへの展望**: Nintendo Switch で遊べる未来を本気で目指しています。
