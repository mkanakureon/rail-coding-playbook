# エディタ機能拡張案：デバッグと AI 連携の深化

**作成日**: 2026-02-26
**対象**: `apps/editor` の機能強化

## 1. 変数・フラグ監視パネル (Inspector)
- **機能**: プレビュー実行中に、現在の全てのグローバル変数およびローカル変数をリアルタイムでリスト表示します。
- **目的**: 複雑な分岐条件（if文）が意図通りに動いているかを検証しやすくします。

## 2. AI 生成の「Human-in-the-Loop」統合
- **課題**: 現在は .md から一気に .ks まで生成されますが、途中のプロット（Stage 1/2）で AI が間違えた場合、最終的な本文がすべてズレてしまいます。
- **拡張**:
    - Stage 1（章プロット）が終わった時点で生成を一時停止し、エディタ上でユーザーが編集できる画面を追加。
    - ユーザーが承認（Approve）ボタンを押すと、その「修正されたプロット」を元に Stage 2 へ進む。

## 3. アセット自動マッピング・サジェスト
- **機能**: シナリオ（KSC）内のト書きを AI が解析し、「ここは forest_evening が最適です」とアセットを自動サジェスト、または自動で `bg()` コマンドを挿入する機能。

## 4. 結論
これらの拡張により、kaedevn は単なる「エディタ」から「AI と人間が協調して作品を磨き上げる統合開発環境」へと進化します。
