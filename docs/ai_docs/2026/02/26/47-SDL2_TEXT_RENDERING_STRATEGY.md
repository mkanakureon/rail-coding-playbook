# SDL2 テキスト描画戦略：メッセージウィンドウの実装

**作成日**: 2026-02-26
**対象**: `SDL_ttf` を用いた文字表示ロジック
**核心**: 日本語フォントの表示とメッセージ送りの実現

## 1. 描画コンポーネント
- **FontManager**: `.ttf` / `.otf` フォントをロードし、`TTF_Font*` を管理。
- **MessageWindow**: 画面下部に配置される、半透明の背景矩形。
- **TextSurface**: セリフを `SDL_Surface` (RenderText_Blended_Wrapped) 経由で `SDL_Texture` へ変換し、描画。

## 2. 禁則処理とラッピング
- `TTF_RenderUTF8_Blended_Wrapped` を使用し、指定した幅（例: 1100px）で自動改行を行う。
- 文字送り演出（タイピング効果）は、表示する文字列の長さを時間経過（`deltaTime`）で変化させることで実現。

## 3. 実装の優先順位
1.  **静的表示**: 全文を一気に表示する。
2.  **動的表示**: 1文字ずつ表示し、クリックで全文表示にスキップ。
3.  **複数フォント**: 名前（太字）とセリフ（標準）の切り替え。
