# スクリーンショット撮影機能：仕様・操作ガイド

**作成日**: 2026-02-26
**対象**: ユーザー・テスター
**核心**: 現在のゲーム画面を PNG 画像として保存する

## 1. 概要
ネイティブエンジン実行中に、ボタン一発で画面全体をキャプチャし、高品質な PNG ファイルとして保存します。開発時のバグ報告や、SNS へのシェア用画像作成に活用できます。

## 2. 操作方法 (Trigger)
- **Keyboard**: `P` キー。
- **Gamepad (将来対応)**: キャプチャボタン または R3 ボタン。

## 3. 保存仕様
- **フォーマット**: PNG (無劣化圧縮)。
- **解像度**: ウィンドウの論理サイズ（標準 1280x720）に基づきます。
- **ファイル名規則**: `screenshot_YYYYMMDD_HHMMSS.png`。
- **保存場所**: 実行バイナリと同じディレクトリ。

## 4. 技術的な仕組み
1.  `SDL_RenderReadPixels` を使用して、GPU 上のバックバッファからピクセルデータを読み取ります。
2.  読み取ったデータを `SDL_Surface` に変換します。
3.  `IMG_SavePNG` (SDL_image) を用いて、非同期（メインスレッドを止めない形式）でディスクへ書き出します。
