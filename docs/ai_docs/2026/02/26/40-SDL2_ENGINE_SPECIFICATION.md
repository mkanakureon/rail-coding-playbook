# SDL2 ネイティブエンジン：機能仕様書

**作成日**: 2026-02-26
**対象**: C++ 実装エンジニア
**核心**: Switch で Kaede Script (.ksc) を安定実行するための最小要件

## 1. システム要件
- **ターゲット**: Nintendo Switch (API レベル 1.0+), Windows 10/11.
- **グラフィックライブラリ**: SDL2 (SDL_Renderer 使用).
- **言語**: C++17 以上。

## 2. 描画仕様 (Rendering)
- **解像度**: 1280x720 (Logical).
- **アセット形式**: PNG (アルファチャンネル付き).
- **レイヤー構造**:
    1.  **Background**: 最背面、全画面表示。
    2.  **Character (N個)**: 透過立ち絵、位置指定（x, y）。
    3.  **UI/Message**: テキストウィンドウ、ボタン。
- **トランジション**: `cross-fade` を必須サポート。テクスチャの alpha 値をフレームごとに増減。

## 3. 音響仕様 (Audio)
- **ライブラリ**: SDL_mixer.
- **チャンネル管理**:
    - `CH_BGM`: ループ再生、フェードイン/アウト対応。
    - `CH_SE`: 単発再生、複数同時再生可能。
    - `CH_VOICE`: キャラクター音声、SE とは別にボリューム制御。

## 4. 入力仕様 (Input)
`IInput` インターフェースに基づき、以下のマッピングを必須とします。

| Action | Switch ボタン | 役割 |
| :--- | :--- | :--- |
| **`OK`** | A ボタン | メッセージ送り、選択決定 |
| **`BACK`** | B ボタン | キャンセル、メニューを閉じる |
| **`MENU`** | + ボタン | システムメニュー表示 |
| **`SKIP`** | R ボタン押下中 | 高速スキップ |

## 5. スクリプト実行
- **Kaede Script**: TypeScript 版インタプリタと同一の文法を解釈すること。
- **変数**: 整数、浮動小数、文字列、真偽値のグローバル変数を保持・復元可能であること。
