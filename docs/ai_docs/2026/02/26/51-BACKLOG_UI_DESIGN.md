# バックログ（履歴）画面：UI/UX 設計仕様

**作成日**: 2026-02-26
**対象**: プレイ体験の向上
**核心**: 「読み返し」と「聞き直し」のシームレスな提供

## 1. 起動トリガー
- **Mouse**: ホイール上回転。
- **Keyboard**: PageUp キー または L キー。
- **Gamepad**: L ボタン または 十字キー上。

## 2. 画面構成 (Visual Layout)
- **背景**: ゲーム画面を暗く透過（Black 70% 程度）させたオーバーレイ。
- **リストアイテム**:
    - **話者名**: 左側に色付きで表示。
    - **セリフ本文**: 右側に折り返して表示。
    - **音声アイコン**: ボイスが存在する場合、クリックで再再生。
- **スクロール**: スムーズな慣性スクロール（PixiJS/SDL2 両対応）。

## 3. データ保持仕様
- **最大件数**: 直近 100 件〜200 件。
- **永続化**: セーブデータには含めず、現在のセッション（起動中）のみ保持。

## 4. 特殊アクション
- **ジャンプ (将来拡張)**: ログの特定の箇所をクリックすると、そのシーンまで物語を引き戻す（巻き戻し実行）機能の検討。
