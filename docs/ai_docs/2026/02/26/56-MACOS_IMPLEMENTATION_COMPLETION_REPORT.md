## 1. 達成されたマイルストーン

### 1.1 レンダリング・演出基盤
- **HighDPI 対応**: Retina ディスプレイ環境での論理 1280x720 解像度を維持。
- **グラフィック**: 背景、立ち絵（精密なX座標指定）、メッセージウィンドウの実装。
- **シェーダー基盤**: GLSL コンパイルと Uniform 連携の仕組み（将来のマスクワイプ用）。

### 1.2 スクリプト・制御ロジック
- **KSC 互換性**: TS 版と同一の文法を解釈可能な C++ 版インタプリタ。
- **高度なフロー制御**: `if` 分岐、`choice` 選択肢、サブルーチン（call/ret）。
- **プレイ QOL**: 高速スキップ、セーブ・ロード（スナップショット方式）、バックログ表示。

### 1.3 開発効率 (DX)
- **RAII 管理**: スマートポインタによる 100% メモリリークのない設計。
- **CLI ツール**: 引数解析による自由なスクリプト指定、自動ビルド＆テストスクリプト。
- **データ同期**: エディタの出力をワンコマンドで反映する `sync-native.sh`。
