# ビジュアル演出強化ロードマップ：商用クオリティへの道

**作成日**: 2026-02-26
**対象**: レンダリングエンジンの高度化

## 1. 高度なトランジション (Rich Transitions)
- **現状**: opacity によるクロスフェードのみ。
- **目標**: PixiJS のカスタムシェーダーを使用し、ルール画像（mask）を用いた多様なワイプ演出（例：桜吹雪状に消える、波紋状に現れる）をサポート。
- **KSC拡張**: `bg("slug", "wipe:mask_flower", 1000)`

## 2. 動的エフェクトとシェーダー
- **スクリーンフィルタ**: 全画面に適用するセピア、モノクロ、走査線、ブラー（ボケ）フィルタの標準搭載。
- **パーティクルシステム**: 舞い落ちる雪、雨、火の粉などを `packages/web` の組み込み演出として提供。

## 3. キャラクター演出の深化
- **目パチ・口パク**: `FrameSet` 機能を応用し、セリフ表示中にランダムまたは音量（VOICE）同期で表情の一部を差し替えるロジックの追加。
- **位置アニメーションのプリセット**: 「軽く跳ねる」「震える」「背後に下がる」といった VN 特有の動記（ジェスチャー）をコマンド一つで呼び出せるようにする。

## 4. 優先実装事項 (Quick Wins)
1.  **シェーダーベースのトランジション**: 演出の幅が劇的に広がります。
2.  **全画面フィルタ**: 回想シーン等の雰囲気作りに不可欠です。
