## 2. 全ブロック（コマンド）別 詳細検証項目

### 2.1 テキスト・基本演出
- **[text] セリフ**: 
    - [ ] `body`: テキスト内容の編集と改行の保持。
    - [ ] `who`: キャラクター名の指定と、未指定時の挙動。
- **[bg] 背景変更**:
    - [ ] `id`: アセットセレクタからの画像選択とプレビュー。
    - [ ] `fadeMs`: フェード時間の数値バリデーション（負数禁止）。
    - [ ] `effect`: `fade`, `slide_left`, `slide_right` の各エフェクト選択の反映。
- **[ch] キャラ表示**:
    - [ ] `name`, `pose`: キャラクター名とポーズ（立ち絵）の選択。
    - [ ] `pos`: `left`, `center`, `right` の座標プリセットの反映。

### 2.2 アニメーション・高度なビジュアル
- **[live2d_ch] Live2D**:
    - [ ] `modelId`: モデル（Hiyori, Mao 等）の切り替え。
    - [ ] `motionGroup`, `motionIndex`: 選択したモデルに存在するモーションのみがリストに出るか。
- **[timeline] タイムライン**:
    - [ ] `tracks`: `camera`, `entity`, `audio` トラックの追加・削除。
    - [ ] `keyframes`: キーフレームの作成、時間軸上の移動、値の設定。
- **[camera] カメラ**:
    - [ ] `zoom`, `shake`: ズーム倍率と揺れ強度の数値設定。
    - [ ] `time`, `easing`: 移動時間とイージングカーブの選択。
- **[particle] パーティクル**:
    - [ ] `preset`: `snow`, `rain`, `fire` 等のプリセット選択と強度の反映。

### 2.3 画面フィルタ・色調補正
- **[screen_filter] フィルタ**:
    - [ ] プリセット（セピア、モノクロ、ピクセル化）の単一適用。
- **[filter_mix] フィルタ合成**:
    - [ ] `layers`: 2〜4層のフィルタレイヤーの追加・順序入れ替え。
    - [ ] `intensity`: 各レイヤーごとの適用強度の個別設定。
- **[color_adjust] 色調補正**:
    - [ ] `brightness`, `contrast`, `saturation`, `temperature` の各スライダー操作。

### 2.4 ロジック・変数・分岐
- **[set_var] 変数設定**:
    - [ ] `op`: `=`, `+=`, `-=` の演算子選択。
    - [ ] `name`, `value`: 変数名と代入値（数値/文字列）の設定。
- **[choice] 選択肢**:
    - [ ] `options`: 選択肢の追加、各項目のラベル、ジャンプ先ラベルの設定。
- **[if] 条件分岐**:
    - [ ] `condition`: 条件式の記述。
    - [ ] `then/else`: それぞれのブランチ内へのブロック追加とネスト管理。

### 2.5 その他
- **[ksc] KSコード**:
    - [ ] 直接記述した KAG スクリプトの保存と、シンタックスエラーの表示。
- **[overlay] オーバーレイ**:
    - [ ] 全面被せ画像（ノイズ、雨等）の透過度設定。
